## 95.01 — Prometheus + Grafana Setup

**Exposing metrics**
```csharp
// NuGet: prometheus-net.AspNetCore, OpenTelemetry.Exporter.Prometheus.AspNetCore

// Expose metrics endpoint
app.MapPrometheusScrapingEndpoint("/metrics");

// Default metrics (auto-included):
// - http_requests_total (counter: total requests by method, path, status)
// - http_request_duration_seconds (histogram: latency by method, path)
// - dotnet_gc_collections_total (GC collections by generation)
// - dotnet_memory_working_set_bytes (process memory)
// - process_cpu_seconds_total (CPU usage)

// Custom business metrics
public class OrderMetrics
{
    private readonly Counter<int> _ordersCreated;
    private readonly Histogram<double> _orderProcessingTime;
    private readonly ObservableGauge<int> _activeOrders;
    private int _activeOrderCount;
    
    public OrderMetrics(IMeterFactory meterFactory)
    {
        var meter = meterFactory.Create("MyApp.Orders");
        
        _ordersCreated = meter.CreateCounter<int>(
            "orders_created_total",
            description: "Total number of orders created");
        
        _orderProcessingTime = meter.CreateHistogram<double>(
            "order_processing_duration_seconds",
            unit: "s",
            description: "Order processing time");
        
        _activeOrders = meter.CreateObservableGauge<int>(
            "orders_active",
            () => _activeOrderCount,
            description: "Currently active orders");
    }
    
    public void RecordOrderCreated(string status) =>
        _ordersCreated.Add(1, new KeyValuePair<string, object?>("status", status));
    
    public void RecordProcessingTime(double seconds, string orderType) =>
        _orderProcessingTime.Record(seconds,
            new KeyValuePair<string, object?>("type", orderType));
    
    public void SetActiveOrderCount(int count) =>
        Interlocked.Exchange(ref _activeOrderCount, count);
}
```

**Essential Grafana dashboards**

**Dashboard 1: API Health**
```
Panels:
- Request rate (req/s) → rate(http_requests_total[5m])
- Error rate (%) → rate(http_requests_total{status=~"5.."}[5m]) / rate(http_requests_total[5m]) * 100
- P50/P95/P99 latency → histogram_quantile(0.99, rate(http_request_duration_seconds_bucket[5m]))
- Active connections → http_connections_current
```

**Dashboard 2: .NET Runtime Health**
```
Panels:
- GC Gen0/1/2 collection rate → rate(dotnet_gc_collections_total[5m])
- Memory usage → dotnet_memory_working_set_bytes
- Thread pool threads → dotnet_threadpool_threads_count
- Exception rate → rate(dotnet_exceptions_total[5m])
```

**Dashboard 3: Business Metrics**
```
Panels:
- Orders created/hour → increase(orders_created_total[1h])
- Order success rate → rate(orders_created_total{status="success"}[5m]) / rate(orders_created_total[5m])
- Active orders → orders_active
- Revenue/hour → increase(payment_amount_total[1h])
```

## 95.02 — Alerting Rules (PagerDuty/Opsgenie)

**Prometheus AlertManager rules**
```yaml
groups:
- name: api-alerts
  rules:
  
  # High error rate
  - alert: HighErrorRate
    expr: |
      rate(http_requests_total{status=~"5.."}[5m])
      / rate(http_requests_total[5m]) * 100 > 5
    for: 2m  # Must be true for 2 minutes (avoids transient spikes)
    labels:
      severity: critical
    annotations:
      summary: "High error rate: {{ $value | printf \"%.1f\" }}%"
      description: "Error rate is {{ $value | printf \"%.1f\" }}% (threshold: 5%)"
      runbook: "https://wiki.example.com/runbooks/high-error-rate"
  
  # High latency
  - alert: HighLatency
    expr: |
      histogram_quantile(0.99, rate(http_request_duration_seconds_bucket[5m])) > 2
    for: 5m
    labels:
      severity: warning
    annotations:
      summary: "P99 latency high: {{ $value | printf \"%.2f\" }}s"
  
  # Low disk space
  - alert: LowDiskSpace
    expr: disk_free_bytes / disk_total_bytes * 100 < 10
    for: 1m
    labels:
      severity: critical
    annotations:
      summary: "Disk space critical: {{ $value | printf \"%.1f\" }}% remaining"
  
  # High memory
  - alert: HighMemoryUsage
    expr: dotnet_memory_working_set_bytes / 1024 / 1024 > 1500  # 1.5 GB
    for: 10m
    labels:
      severity: warning
    annotations:
      summary: "Memory usage: {{ $value | printf \"%.0f\" }}MB"
  
  # No traffic (possible outage)
  - alert: NoTraffic
    expr: rate(http_requests_total[5m]) == 0
    for: 5m
    labels:
      severity: critical
    annotations:
      summary: "No incoming traffic for 5 minutes — possible outage"
```
