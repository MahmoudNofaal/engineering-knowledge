## 85.01 — Complete CQRS Implementation

**What CQRS is**
Command Query Responsibility Segregation: separate the write model (commands) from the read model (queries).

**Level 1: Same database, different models**
```csharp
// Write side: rich domain model
// Read side: flat DTOs optimized for queries

// Write: uses the domain model
public class CreateOrderHandler : IRequestHandler<CreateOrderCommand, int>
{
    private readonly IOrderRepository _repo;
    private readonly IUnitOfWork _uow;
    
    public async Task<int> Handle(CreateOrderCommand cmd, CancellationToken ct)
    {
        var order = Order.Create(cmd.CustomerId, cmd.Items.Select(i =>
            OrderItem.Create(i.ProductId, i.Quantity, i.Price)));
        
        _repo.Add(order);
        await _uow.SaveChangesAsync(ct);
        return order.Id;
    }
}

// Read: uses optimized queries (bypasses domain model)
public class GetOrdersQueryHandler : IRequestHandler<GetOrdersQuery, PagedResult<OrderSummaryDto>>
{
    private readonly AppDbContext _db;
    
    public async Task<PagedResult<OrderSummaryDto>> Handle(GetOrdersQuery query, CancellationToken ct)
    {
        // Direct SQL-like query, no domain model involved
        return await _db.Orders
            .AsNoTracking()
            .Where(o => o.CustomerId == query.CustomerId)
            .Select(o => new OrderSummaryDto
            {
                Id = o.Id,
                Status = o.Status.ToString(),
                Total = o.Total,
                ItemCount = o.Items.Count,
                CustomerName = o.Customer.Name,
                CreatedAt = o.CreatedAt
            })
            .OrderByDescending(o => o.CreatedAt)
            .ToPagedResultAsync(query.Page, query.PageSize, ct);
    }
}
```

**Level 2: Separate read database (with event sync)**
```csharp
// Write to: SQL Server (normalized, consistent)
// Read from: PostgreSQL / Redis / Elasticsearch (denormalized, fast)
// Sync: via domain events → update read model

public class OrderCreatedReadModelUpdater : INotificationHandler<OrderCreatedEvent>
{
    private readonly ReadDbContext _readDb;  // Separate read database
    
    public async Task Handle(OrderCreatedEvent evt, CancellationToken ct)
    {
        // Denormalized read model — optimized for the common query pattern
        await _readDb.OrderReadModels.AddAsync(new OrderReadModel
        {
            Id = evt.OrderId,
            CustomerId = evt.CustomerId,
            CustomerName = evt.CustomerName,  // Denormalized — no join needed!
            CustomerEmail = evt.CustomerEmail,
            Status = "Pending",
            Total = evt.Total,
            ItemCount = evt.ItemCount,
            CreatedAt = evt.CreatedAt,
            // Pre-aggregated fields for dashboards
            StatusForDashboard = "🕐 Pending"
        }, ct);
        
        await _readDb.SaveChangesAsync(ct);
    }
}

public class OrderShippedReadModelUpdater : INotificationHandler<OrderShippedEvent>
{
    private readonly ReadDbContext _readDb;
    
    public async Task Handle(OrderShippedEvent evt, CancellationToken ct)
    {
        var model = await _readDb.OrderReadModels.FindAsync(evt.OrderId, ct);
        if (model is null) return;
        
        model.Status = "Shipped";
        model.ShippedAt = evt.ShippedAt;
        model.TrackingNumber = evt.TrackingNumber;
        model.StatusForDashboard = "🚚 Shipped";
        
        await _readDb.SaveChangesAsync(ct);
    }
}
```

**Level 3: Event Sourcing + CQRS**
- Commands → domain events → append to event store
- Projections listen to event store → build read models
- Multiple projections from same events (dashboard view, API view, analytics view)

```csharp
// Projection rebuilding (replay all events to rebuild read model)
public class ProjectionRebuilder
{
    private readonly IEventStore _events;
    private readonly IReadModelStore _readModels;
    
    public async Task RebuildOrderProjectionAsync(CancellationToken ct)
    {
        // Clear current read model
        await _readModels.ClearOrdersAsync(ct);
        
        // Replay all events in order
        await foreach (var @event in _events.ReadAllAsync(ct))
        {
            await ApplyToProjection(@event, ct);
        }
    }
}
```
