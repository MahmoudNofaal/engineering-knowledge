## 24.01 — BenchmarkDotNet

**What it is**
The standard .NET library for accurate micro-benchmarks. Eliminates common benchmarking mistakes (JIT warmup, GC interference, measurement overhead).

**Basic benchmark**
```csharp
[MemoryDiagnoser]
[SimpleJob(RuntimeMoniker.Net80)]
public class StringConcatenationBenchmarks
{
    private const int Iterations = 1000;
    
    [Benchmark(Baseline = true)]
    public string PlusOperator()
    {
        var result = "";
        for (int i = 0; i < Iterations; i++)
            result += i.ToString();
        return result;
    }
    
    [Benchmark]
    public string StringBuilder()
    {
        var sb = new StringBuilder();
        for (int i = 0; i < Iterations; i++)
            sb.Append(i);
        return sb.ToString();
    }
    
    [Benchmark]
    public string StringCreate()
    {
        // Most efficient for known-length strings
        return string.Concat(Enumerable.Range(0, Iterations).Select(i => i.ToString()));
    }
}

// Run:
// BenchmarkRunner.Run<StringConcatenationBenchmarks>();
```

**Results interpretation**
```
| Method        | Mean       | Allocated |
|---------------|------------|-----------|
| PlusOperator  | 1,234 μs   | 2,048 KB  | ← Baseline: allocates many strings
| StringBuilder |    45 μs   |    16 KB  | ← 27x faster, 128x less memory
| StringCreate  |    38 μs   |     8 KB  | ← Best: fastest, least memory
```

## 24.02 — Memory Optimization Patterns

**Object Pooling**
```csharp
// Using Microsoft.Extensions.ObjectPool
builder.Services.AddSingleton(serviceProvider =>
{
    var provider = new DefaultObjectPoolProvider();
    return provider.Create(new DefaultPooledObjectPolicy<StringBuilder>());
});

public class ReportGenerator
{
    private readonly ObjectPool<StringBuilder> _pool;
    
    public string GenerateReport(IEnumerable<Order> orders)
    {
        var sb = _pool.Get();
        try
        {
            sb.Clear();
            foreach (var order in orders)
                sb.AppendLine($"Order {order.Id}: {order.Total:C}");
            return sb.ToString();
        }
        finally
        {
            _pool.Return(sb);
        }
    }
}
```

**ArrayPool**
```csharp
// Rent and return buffers from pool
byte[] buffer = ArrayPool<byte>.Shared.Rent(4096);
try
{
    int bytesRead = await stream.ReadAsync(buffer.AsMemory(0, buffer.Length));
    await ProcessData(buffer.AsSpan(0, bytesRead));
}
finally
{
    ArrayPool<byte>.Shared.Return(buffer, clearArray: false);
}
```

**Span for zero-copy parsing**
```csharp
// Parse CSV line without allocating substrings
ReadOnlySpan<char> line = "Alice,30,Engineer".AsSpan();

int comma1 = line.IndexOf(',');
var name = line[..comma1];

line = line[(comma1 + 1)..];
int comma2 = line.IndexOf(',');
var ageSpan = line[..comma2];
int age = int.Parse(ageSpan);

var role = line[(comma2 + 1)..];
```

## 24.03 — Database Performance

**Dapper for performance-critical reads**
```csharp
// Dapper: thin ORM, close to raw SQL performance
public async Task<IEnumerable<OrderSummary>> GetOrderSummariesAsync(int customerId)
{
    using var connection = new SqlConnection(_connectionString);
    
    return await connection.QueryAsync<OrderSummary>("""
        SELECT
            o.Id,
            o.OrderDate,
            o.Status,
            SUM(oi.Price * oi.Quantity) AS Total,
            COUNT(oi.Id) AS ItemCount
        FROM Orders o
        INNER JOIN OrderItems oi ON o.Id = oi.OrderId
        WHERE o.CustomerId = @CustomerId
        GROUP BY o.Id, o.OrderDate, o.Status
        ORDER BY o.OrderDate DESC
        """,
        new { CustomerId = customerId });
}
```

**EF Core compiled queries**
```csharp
// Compile once, reuse — eliminates per-call query translation overhead
private static readonly Func<AppDbContext, int, Task<Order?>> GetOrderByIdQuery =
    EF.CompileAsyncQuery((AppDbContext db, int orderId) =>
        db.Orders
            .Include(o => o.Items)
            .FirstOrDefault(o => o.Id == orderId));

// Usage
var order = await GetOrderByIdQuery(_db, orderId);
```
