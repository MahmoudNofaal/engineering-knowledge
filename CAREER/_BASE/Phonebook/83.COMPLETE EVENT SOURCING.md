## 83.01 — Event Sourcing Pattern

**What it is**
Instead of storing current state, store the sequence of events that led to current state.

```
Traditional:
  orders table: {id: 42, status: "Shipped", total: 99.99, ...}
  ← Single row. History lost. No audit trail.

Event Sourced:
  order_events table:
    {orderId: 42, eventType: "OrderCreated", data: {...}, timestamp: T1}
    {orderId: 42, eventType: "ItemAdded", data: {product: 1, qty: 2}, timestamp: T2}
    {orderId: 42, eventType: "DiscountApplied", data: {percent: 10}, timestamp: T3}
    {orderId: 42, eventType: "PaymentReceived", data: {amount: 89.99}, timestamp: T4}
    {orderId: 42, eventType: "OrderShipped", data: {trackingNo: "ABC123"}, timestamp: T5}
  
  Current state = replay of all events
```

**When to use**
- When audit trail is required (financial, medical, legal systems)
- When you need time-travel (show what state was at any point in time)
- When business events have intrinsic value beyond state changes
- Complex domain with many state transitions

**When NOT to use**
- Simple CRUD with no real domain events
- When querying current state is the primary operation
- Small teams that don't have the expertise to operate it

**Implementation**
```csharp
// Event base
public abstract record DomainEvent
{
    public Guid Id { get; } = Guid.NewGuid();
    public DateTime OccurredOn { get; } = DateTime.UtcNow;
    public int Version { get; init; }
}

// Events
public record OrderCreatedEvent(int OrderId, int CustomerId, DateTime CreatedAt)
    : DomainEvent;
public record ItemAddedEvent(int OrderId, int ProductId, int Quantity, decimal Price)
    : DomainEvent;
public record OrderShippedEvent(int OrderId, string TrackingNumber, DateTime ShippedAt)
    : DomainEvent;

// Aggregate rebuilt from events
public class Order
{
    public int Id { get; private set; }
    public int CustomerId { get; private set; }
    public OrderStatus Status { get; private set; }
    public List<OrderItem> Items { get; private set; } = new();
    public int Version { get; private set; }
    
    // Apply events to reconstruct state (pure functions, no side effects)
    private void Apply(OrderCreatedEvent e)
    {
        Id = e.OrderId;
        CustomerId = e.CustomerId;
        Status = OrderStatus.Pending;
    }
    
    private void Apply(ItemAddedEvent e)
    {
        Items.Add(new OrderItem(e.ProductId, e.Quantity, e.Price));
    }
    
    private void Apply(OrderShippedEvent e)
    {
        Status = OrderStatus.Shipped;
    }
    
    // Rebuild from event stream
    public static Order Reconstitute(IEnumerable<DomainEvent> events)
    {
        var order = new Order();
        foreach (var @event in events)
        {
            order.Version = @event.Version;
            order.Apply((dynamic)@event);  // Dynamic dispatch to correct Apply overload
        }
        return order;
    }
}

// Event store
public interface IEventStore
{
    Task AppendEventsAsync(int aggregateId, IEnumerable<DomainEvent> events,
        int expectedVersion, CancellationToken ct);
    Task<IReadOnlyList<DomainEvent>> LoadEventsAsync(int aggregateId, CancellationToken ct);
    Task<IReadOnlyList<DomainEvent>> LoadEventsAsync(int aggregateId,
        int fromVersion, int toVersion, CancellationToken ct);
}
```

**Snapshotting (performance optimization)**
```csharp
// After N events, store a snapshot of current state
// Next load: load snapshot + events after snapshot version

public class SnapshotStore
{
    public async Task SaveSnapshotAsync(Order order, CancellationToken ct)
    {
        var snapshot = new OrderSnapshot
        {
            OrderId = order.Id,
            State = JsonSerializer.Serialize(order),
            Version = order.Version,
            CreatedAt = DateTime.UtcNow
        };
        await _db.Snapshots.AddAsync(snapshot, ct);
        await _db.SaveChangesAsync(ct);
    }
    
    public async Task<Order?> LoadSnapshotAsync(int orderId, CancellationToken ct)
    {
        var snapshot = await _db.Snapshots
            .Where(s => s.OrderId == orderId)
            .OrderByDescending(s => s.Version)
            .FirstOrDefaultAsync(ct);
        
        return snapshot is null
            ? null
            : JsonSerializer.Deserialize<Order>(snapshot.State);
    }
}

// Repository
public async Task<Order?> GetOrderAsync(int orderId, CancellationToken ct)
{
    var snapshot = await _snapshots.LoadSnapshotAsync(orderId, ct);
    
    if (snapshot is not null)
    {
        // Load only events after snapshot version
        var newEvents = await _eventStore.LoadEventsAsync(
            orderId, fromVersion: snapshot.Version + 1, toVersion: int.MaxValue, ct);
        
        return Order.Reconstitute(newEvents, snapshot);  // Start from snapshot
    }
    
    var allEvents = await _eventStore.LoadEventsAsync(orderId, ct);
    return allEvents.Any() ? Order.Reconstitute(allEvents) : null;
}
```
