# Career & Job Search Roadmap

## Vision

Become a professional Backend Software Engineer specializing in ASP.NET Core and modern backend systems, securing a full-time software engineering role and building a strong long-term career in software development.

---

# Current Position

## Profile

- Fresh Graduate
- ~2 Years Backend / Software Engineering Experience
- ASP.NET Core Developer
- C# Developer
- Strong Learning Discipline
- Building Deep ASP.NET Core Knowledge Base

## Target Roles

### Primary

- Backend Software Engineer (.NET)
- Junior Software Engineer (.NET)
- ASP.NET Core Developer
- Backend Developer
- Software Engineer

### Secondary

- Associate Software Engineer
- Graduate Software Engineer
- Backend Engineer

### Avoid Positioning As

- Intern
- Trainee
- Fresh Graduate Only

Experience matters more than graduation status.

## Main Goal

Secure a Backend Software Engineering position.
