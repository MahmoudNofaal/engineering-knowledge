Enforces constant output rate. Excess requests are queued (bounded buffer). When buffer fills, overflow = rejection. Used for traffic shaping: video encoding, database write throttling.

```csharp
public class LeakyBucket {
    private readonly Channel<Func<ValueTask>> _queue;
    private readonly int _ratePerSecond;
    public bool TryEnqueue(Func<ValueTask> work) => _queue.Writer.TryWrite(work);
    public async Task DrainAsync(CancellationToken ct) {
        await foreach (var work in _queue.Reader.ReadAllAsync(ct)) {
            await work();
            await Task.Delay(1000 / _ratePerSecond, ct);
        }
    }
}
```

**Key:** No burst. Constant output. Overflow = drop. Monitor queue depth.

**ADR:** Video transcoding at 5 fps. Input spikes to 50 fps. Leaky bucket capacity 30 frames, drain 5 fps. ✅ Constant output; ⚠️ Monitor depth; ❌ Drops above sustained 5 fps.
