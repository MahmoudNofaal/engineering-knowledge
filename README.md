# Engineering Knowledge Base

> My personal engineering library. Written in my own words, built from real experience.
> Every topic follows the same template. Open any file — you always know where everything is.

---

## How To Use This

- **Before an interview:** open the domain folder, scan the topic files, read the *Interview Angle* section of anything relevant.
- **Learning something new:** copy `_template.md`, name it `kebab-case.md`, put it in the right folder, fill in what you understand *right now*. Incomplete is fine. Copied-from-docs is not.
- **Revisiting a topic:** update the file in place, change the `Last updated` date, commit.
- **Commit messages:** `add: topic-name` / `update: topic-name` / `fix: topic-name`
