# 8.964 Redis — Strings — APPEND, STRLEN, GETRANGE, SETRANGE

## Section 1 — Overview

Redis Strings are binary-safe byte sequences. Beyond basic SET/GET and atomic counters, Redis provides string manipulation commands: APPEND concatenates to the end, STRLEN returns the byte length, GETRANGE extracts a substring by byte positions, and SETRANGE overwrites bytes at a given offset. These commands operate at the byte level, which is critical to understand when working with multi-byte character encodings like UTF-8.

### Core Commands

| Command | Signature | O-complexity | Description |
|---------|-----------|-------------|-------------|
| APPEND | `APPEND key value` | O(1) — amortized (grows string, may reallocate) | Appends the value to the string at key. Creates key if it does not exist. Returns the new length of the string. |
| STRLEN | `STRLEN key` | O(1) | Returns the byte length of the string value at key. Returns 0 if key does not exist. |
| GETRANGE | `GETRANGE key start end` | O(N) — N is length of returned substring | Returns the substring of the string value, identified by byte range (start and end inclusive). Negative offsets count from the end of the string. |
| SETRANGE | `SETRANGE key offset value` | O(1) — amortized, not counting size of side effects | Overwrites part of the string at the specified byte offset. Creates or pads the string with null bytes if offset is beyond the current length. Returns the new length of the string. |

### Byte-Level vs Character-Level Operations

Redis does not understand character encoding. All string operations work on raw bytes:

```
String: "café"
Bytes:  63 61 66 C3 A9
        c  a  f  é
        (1) (1) (1) (2) = 5 bytes for 4 characters

GETRANGE 0 3 → "caf"   (bytes 0-3: 63 61 66 C3)
GETRANGE 0 2 → "caf"   (bytes 0-2: 63 61 66) — safe cut
GETRANGE 0 4 → "café"  (bytes 0-4: 63 61 66 C3 A9) — complete character
GETRANGE 0 3 → "caf\xC3" — byte 3 is the first byte of 'é', produced invalid UTF-8
```

### When to Use These Commands

| Use Case | Command | Reason |
|----------|---------|--------|
| Append to log stream | APPEND | O(1) amortized, efficient for log aggregation |
| Check string size before processing | STRLEN | O(1) — avoids loading full string just for size |
| Read first N bytes of a large string | GETRANGE | Avoids loading the entire 512 MB string |
| Partial update of a string (e.g., patch) | SETRANGE | Modify a portion without reading-modify-writing full value |
| Implement cursor-based pagination | GETRANGE | Extract pages of data from a large string |
| File-like seek and write | SETRANGE | Overwrite at arbitrary position |

## Section 2 — Command Reference

### APPEND — Append to String

```
APPEND key value

Time complexity: O(1) amortized. The command requires reallocation of the
internal SDS (Simple Dynamic String) when the string grows beyond allocated capacity.
Returns: the length of the string after the append operation.

Behavior:
- If key does not exist, it is created as if SET key "" (empty string) first, then value is appended
- If key exists but does not hold a string value, returns WRONGTYPE error
- Appends value to the existing string value
- Returns the new length of the combined string
- Maximum final length is 512 MB (Redis string limit)
- The value can be any binary data, including null bytes
```

### STRLEN — String Length

```
STRLEN key

Time complexity: O(1)
Returns: the byte length of the string value at key. Returns 0 if key does not exist.

Behavior:
- Returns the length of the string in BYTES, not characters
- For strings with multi-byte characters, the byte length differs from character count
- Returns 0 if key does not exist (not an error)
- Returns WRONGTYPE error if key holds a non-string value
- Useful for capacity checks before reading the entire value
- O(1) because Redis stores string length internally in the SDS header
```

### GETRANGE — Get Substring

```
GETRANGE key start end

Time complexity: O(N) where N is the length of the returned substring.
Returns: the substring specified by the byte range.

Behavior:
- Returns the substring of the string value determined by byte positions start and end
- Both start and end are INCLUSIVE (0-based)
- Negative offsets count from the end of the string: -1 is the last byte
- If start exceeds the string length, returns empty string
- If end exceeds the string length, behaves as if end = length - 1
- Returns the full string when start = 0 and end = -1
- Out-of-range values are clamped to valid range
- Returns empty string if start > end
- Works on BYTE positions — see encoding section for multi-byte considerations
```

### SETRANGE — Set Substring

```
SETRANGE key offset value

Time complexity: O(1) amortized, not counting the size of the side effects.
For large offset + value, requires growing the string and potentially filling with null bytes.
Returns: the length of the string after the operation.

Behavior:
- Overwrites the part of the string starting at the specified byte offset
- If the offset is beyond the current string length, the string is padded with null bytes (\x00)
- If key does not exist, it is created as if SET key "" first, then padded from offset 0
- The value is inserted at the offset; it does NOT shift existing bytes
- If the original string at offset is shorter than value, the string grows
- Returns the new length of the string after the operation
- Maximum string size after operation is 512 MB
- Negative offsets are not allowed (returns error)
```

### Complete Command Set for String Manipulation

| Command | O-complexity | Description |
|---------|-------------|-------------|
| SET | O(1) | Set key to hold string value |
| GET | O(1) | Get value of key |
| APPEND | O(1) amortized | Append value to existing string |
| STRLEN | O(1) | Get byte length of string |
| GETRANGE | O(N) | Get substring by byte range |
| SETRANGE | O(1) amortized | Overwrite part of string at offset |
| GETDEL | O(1) | Get value and delete key |
| GETEX | O(1) | Get value and set expiration |
| SETEX | O(1) | Set with seconds TTL |
| PSETEX | O(1) | Set with milliseconds TTL |
| SETNX | O(1) | Set if not exists |
| MSET | O(N) | Set multiple keys |
| MGET | O(N) | Get multiple keys |
| STRLEN | O(1) | Byte length of string |
| SUBSTR | O(N) | Deprecated alias for GETRANGE |

## Section 3 — Redis CLI Examples

### APPEND — Appending to Strings

```bash
# APPEND on non-existent key — creates key and sets value
127.0.0.1:6379> APPEND log:server "Connection established"
(integer) 22
127.0.0.1:6379> GET log:server
"Connection established"

# APPEND to existing string
127.0.0.1:6379> APPEND log:server "\nUser logged in"
(integer) 37
127.0.0.1:6379> GET log:server
"Connection established\nUser logged in"

# APPEND binary data
127.0.0.1:6379> APPEND binary:log "\x00\x01\x02"
(integer) 3
127.0.0.1:6379> STRLEN binary:log
(integer) 3

# APPEND returns the new length after each operation
127.0.0.1:6379> APPEND log:server "\nProcessing request"
(integer) 55
127.0.0.1:6379> APPEND log:server "\nRequest complete"
(integer) 72

# APPEND with empty string
127.0.0.1:6379> APPEND log:server ""
(integer) 72  # No change in length

# APPEND to a non-string key returns error
127.0.0.1:6379> LPUSH mylist "element"
(integer) 1
127.0.0.1:6379> APPEND mylist "cannot append to list"
(error) WRONGTYPE Operation against a key holding the wrong kind of value
```

### STRLEN — String Byte Length

```bash
# STRLEN on a simple ASCII string
127.0.0.1:6379> SET message "Hello World"
OK
127.0.0.1:6379> STRLEN message
(integer) 11  # 11 bytes = 11 characters (ASCII)

# STRLEN on multi-byte UTF-8 string
127.0.0.1:6379> SET utf8 "café"
OK
127.0.0.1:6379> STRLEN utf8
(integer) 5  # 5 bytes for 4 characters (é is 2 bytes)

# STRLEN on non-existent key returns 0
127.0.0.1:6379> STRLEN nonexistent
(integer) 0

# STRLEN on a large string
127.0.0.1:6379> SET large "x"
OK
127.0.0.1:6379> APPEND large "x"
(integer) 2
127.0.0.1:6379> STRLEN large
(integer) 2

# STRLEN on non-string type returns error
127.0.0.1:6379> SADD myset member1
(integer) 1
127.0.0.1:6379> STRLEN myset
(error) WRONGTYPE Operation against a key holding the wrong kind of value
```

### GETRANGE — Substring Extraction

```bash
# Basic GETRANGE
127.0.0.1:6379> SET greeting "Hello, World!"
OK
127.0.0.1:6379> GETRANGE greeting 0 4
"Hello"
127.0.0.1:6379> GETRANGE greeting 7 11
"World"

# GETRANGE with negative offsets (count from end)
127.0.0.1:6379> GETRANGE greeting -6 -1
"World!"       # Last 6 characters
127.0.0.1:6379> GETRANGE greeting -5 -2
"orld"         # From 5th-to-last to 2nd-to-last

# GETRANGE full string using 0 and -1
127.0.0.1:6379> GETRANGE greeting 0 -1
"Hello, World!"

# GETRANGE with start beyond string length
127.0.0.1:6379> GETRANGE greeting 100 200
""             # Empty string

# GETRANGE with end beyond string length — clamped to end
127.0.0.1:6379> GETRANGE greeting 0 100
"Hello, World!"  # Returns as much as available

# GETRANGE on non-existent key
127.0.0.1:6379> GETRANGE nonexistent 0 10
""              # Empty string

# GETRANGE on multi-byte UTF-8 — byte-level, may split characters
127.0.0.1:6379> SET utf8 "café"
OK
127.0.0.1:6379> GETRANGE utf8 0 2
"caf"           # 3 bytes — safe cut
127.0.0.1:6379> GETRANGE utf8 0 3
"caf\xC3"       # 4 bytes — splits 'é' (2 bytes), result may be invalid UTF-8
127.0.0.1:6379> GETRANGE utf8 0 4
"café"          # 5 bytes — complete character
```

### SETRANGE — Overwrite at Offset

```bash
# Basic SETRANGE — overwrite part of a string
127.0.0.1:6379> SET message "Hello World"
OK
127.0.0.1:6379> SETRANGE message 6 "Redis"
(integer) 11
127.0.0.1:6379> GET message
"Hello Redis"

# SETRANGE at offset 0 (replaces beginning)
127.0.0.1:6379> SETRANGE message 0 "Y"
(integer) 11
127.0.0.1:6379> GET message
"Yello Redis"

# SETRANGE beyond current length — pads with null bytes
127.0.0.1:6379> SET short "abc"
OK
127.0.0.1:6379> SETRANGE short 5 "XYZ"
(integer) 8        # String is now 8 bytes: "abc" + "\x00\x00" + "XYZ"
127.0.0.1:6379> GET short
"abc\x00\x00XYZ"
127.0.0.1:6379> STRLEN short
(integer) 8

# SETRANGE on non-existent key — creates and pads
127.0.0.1:6379> SETRANGE newkey 3 "hello"
(integer) 8        # Created: "\x00\x00\x00hello"
127.0.0.1:6379> GET newkey
"\x00\x00\x00hello"

# SETRANGE with value longer than the overwrite region
127.0.0.1:6379> SET base "XXXXXXXXXX"
OK
127.0.0.1:6379> SETRANGE base 3 "ABCDEF"
(integer) 10       # "XXXABCDEFX" — 6 chars inserted, 1 char remains after
127.0.0.1:6379> GET base
"XXXABCDEFX"

# SETRANGE with exact fit
127.0.0.1:6379> SETRANGE base 3 "ABCDE"
(integer) 10       # "XXXABCDEXX" — 5 chars replace 5 chars
127.0.0.1:6379> GET base
"XXXABCDEXX"
```

```bash

"Title: New Title\nBody: Content here"
```

## Section 4 — StackExchange.Redis Code

### ConnectionMultiplexer Setup

```csharp
using StackExchange.Redis;

/// <summary>
/// Thread-safe singleton for Redis connections.
/// Must be shared application-wide — never create per request.
/// </summary>
public sealed class RedisConnection : IAsyncDisposable
{
    private static readonly Lazy<Task<RedisConnection>> _lazy =
        new Lazy<Task<RedisConnection>>(async () =>
        {
            var conn = new RedisConnection();
            await conn.InitializeAsync();
            return conn;
        });

    private ConnectionMultiplexer _mux;
    private readonly SemaphoreSlim _initLock = new(1, 1);
    private bool _initialized;
    private bool _disposed;

    private RedisConnection() { }

    public static Task<RedisConnection> Instance => _lazy.Value;

    private async Task InitializeAsync()
    {
        if (_initialized) return;
        await _initLock.WaitAsync();
        try
        {
            if (_initialized) return;

            var options = new ConfigurationOptions
            {
                EndPoints = { Environment.GetEnvironmentVariable("REDIS") ?? "localhost:6379" },
                AbortOnConnectFail = false,
                ConnectTimeout = 5000,
                SyncTimeout = 5000,
                ConnectRetry = 3,
                KeepAlive = 60,
                ReconnectRetryPolicy = new ExponentialRetry(1000, 8000),
                ClientName = "RedisStringOps"
            };

            string pwd = Environment.GetEnvironmentVariable("REDIS_PASSWORD");
            if (!string.IsNullOrEmpty(pwd))
                options.Password = pwd;

            _mux = await ConnectionMultiplexer.ConnectAsync(options);

            _mux.ConnectionFailed += (s, e) =>
                Console.Error.WriteLine($"[Redis] FAILED: {e.EndPoint} — {e.Exception?.Message}");

            _mux.ConnectionRestored += (s, e) =>
                Console.WriteLine($"[Redis] RESTORED: {e.EndPoint}");

            _mux.ErrorMessage += (s, e) =>
                Console.Error.WriteLine($"[Redis] Error: {e.Message}");

            _initialized = true;
            Console.WriteLine("[Redis] Connection initialized.");
        }
        finally
        {
            _initLock.Release();
        }
    }

    public IDatabase GetDatabase(int db = -1)
    {
        ObjectDisposedException.ThrowIf(_disposed, this);
        return _mux.GetDatabase(db);
    }

    public async ValueTask DisposeAsync()
    {
        if (_disposed) return;
        _disposed = true;
        if (_mux != null)
        {
            await _mux.CloseAsync();
            _mux.Dispose();
        }
        _initLock.Dispose();
    }
}
```

### IDatabase — APPEND

```csharp
/// <summary>
/// Demonstrates the APPEND command — appends value to string.
/// StackExchange.Redis method: StringAppendAsync
/// Returns the NEW length of the string after append.
/// </summary>
public static class AppendExamples
{
    private static async Task<IDatabase> GetDbAsync()
    {
        var conn = await RedisConnection.Instance;
        return conn.GetDatabase();
    }

    /// <summary>
    /// APPEND — appends value to the string at key.
    /// Creates key if it does not exist.
    /// Returns the new byte length of the string.
    /// </summary>
    public static async Task<long> AppendAsync(string key, string value)
    {
        var db = await GetDbAsync();

        try
        {
            long newLength = await db.StringAppendAsync(key, value);
            Console.WriteLine($"APPEND {key} \"{value}\" → new length: {newLength}");
            return newLength;
        }
        catch (RedisConnectionException ex)
        {
            Console.Error.WriteLine($"Redis connection failed during APPEND {key}: {ex.Message}");
            throw;
        }
        catch (RedisServerException ex) when (ex.Message.Contains("WRONGTYPE"))
        {
            Console.Error.WriteLine($"WRONGTYPE: Key '{key}' is not a string.");
            throw new InvalidOperationException($"Key '{key}' holds a non-string value", ex);
        }
        catch (RedisServerException ex)
        {
            Console.Error.WriteLine($"Redis server error during APPEND {key}: {ex.Message}");
            throw;
        }
        catch (TimeoutException ex)
        {
            Console.Error.WriteLine($"Timeout during APPEND {key}: {ex.Message}");
            throw;
        }
    }

    /// <summary>
    /// APPEND binary data — appends byte array to string.
    /// </summary>
    public static async Task<long> AppendBinaryAsync(string key, byte[] data)
    {
        var db = await GetDbAsync();

        try
        {
            long newLength = await db.StringAppendAsync(key, data);
            Console.WriteLine($"APPEND {key} [{data.Length} bytes] → new length: {newLength}");
            return newLength;
        }
        catch (RedisConnectionException ex)
        {
            Console.Error.WriteLine($"Redis connection failed during APPEND {key}: {ex.Message}");
            throw;
        }
        catch (RedisServerException ex)
        {
            Console.Error.WriteLine($"Redis server error during APPEND {key}: {ex.Message}");
            throw;
        }
        catch (TimeoutException ex)
        {
            Console.Error.WriteLine($"Timeout during APPEND {key}: {ex.Message}");
            throw;
        }
    }

    /// <summary>
    /// Log aggregator using APPEND — appends log lines to a Redis key.
    /// </summary>
    public static async Task<long> AppendLogAsync(string logKey, string logLine)
    {
        var db = await GetDbAsync();

        try
        {
            string formattedLine = $"{DateTime.UtcNow:O} {logLine}\n";
            long newLength = await db.StringAppendAsync(logKey, formattedLine);
            Console.WriteLine($"APPEND {logKey}: +{formattedLine.Length} bytes → total: {newLength}");
            return newLength;
        }
        catch (RedisConnectionException ex)
        {
            Console.Error.WriteLine($"Redis connection failed during log append {logKey}: {ex.Message}");
            // Optionally buffer logs locally if Redis is unavailable
            throw;
        }
        catch (RedisServerException ex)
        {
            Console.Error.WriteLine($"Redis server error during log append {logKey}: {ex.Message}");
            throw;
        }
        catch (TimeoutException ex)
        {
            Console.Error.WriteLine($"Timeout during log append {logKey}: {ex.Message}");
            throw;
        }
    }

    /// <summary>
    /// Checks if APPEND will exceed the 512 MB limit and warns.
    /// </summary>
    public static async Task<bool> TryAppendWithLimitCheckAsync(string key, string value)
    {
        var db = await GetDbAsync();

        try
        {
            long currentLength = await db.StringLengthAsync(key);
            long newLength = currentLength + System.Text.Encoding.UTF8.GetByteCount(value);

            const long maxStringSize = 512 * 1024 * 1024; // 512 MB

            if (newLength > maxStringSize)
            {
                Console.Error.WriteLine($"APPEND would exceed 512 MB limit for key '{key}'");
                Console.Error.WriteLine($"  Current: {currentLength} bytes, Requested: +{value.Length} bytes");
                return false;
            }

            await db.StringAppendAsync(key, value);
            return true;
        }
        catch (RedisConnectionException ex)
        {
            Console.Error.WriteLine($"Redis connection failed: {ex.Message}");
            throw;
        }
        catch (TimeoutException ex)
        {
            Console.Error.WriteLine($"Timeout: {ex.Message}");
            throw;
        }
    }

    /// <summary>
    /// Demonstrates reading intermediate state of an APPEND-built string.
    /// </summary>
    public static async Task BuildAndReadAsync()
    {
        var db = await GetDbAsync();
        string key = "demo:build";

        await db.KeyDeleteAsync(key);

        // Build incrementally
        long len1 = await db.StringAppendAsync(key, "Hello");
        Console.WriteLine($"After APPEND 1: length = {len1}");

        long len2 = await db.StringAppendAsync(key, ", ");
        Console.WriteLine($"After APPEND 2: length = {len2}");

        long len3 = await db.StringAppendAsync(key, "World");
        Console.WriteLine($"After APPEND 3: length = {len3}");

        long len4 = await db.StringAppendAsync(key, "!");
        Console.WriteLine($"After APPEND 4: length = {len4}");

        // Read final result
        RedisValue final = await db.StringGetAsync(key);
        Console.WriteLine($"Final string: \"{final}\"");
        Console.WriteLine($"STRLEN: {await db.StringLengthAsync(key)}");

        // Cleanup
        await db.KeyDeleteAsync(key);
    }
}
```

### IDatabase — STRLEN

```csharp
/// <summary>
/// Demonstrates the STRLEN command — returns byte length of string.
/// StackExchange.Redis method: StringLengthAsync
/// Returns 0 if key does not exist.
/// </summary>
public static class StrLenExamples
{
    private static async Task<IDatabase> GetDbAsync()
    {
        var conn = await RedisConnection.Instance;
        return conn.GetDatabase();
    }

    /// <summary>
    /// STRLEN — gets the byte length of a string value.
    /// Returns 0 if the key does not exist.
    /// </summary>
    public static async Task<long> GetLengthAsync(string key)
    {
        var db = await GetDbAsync();

        try
        {
            long length = await db.StringLengthAsync(key);
            string status = length == 0
                ? "key does not exist or is empty"
                : $"{length} bytes";
            Console.WriteLine($"STRLEN {key} → {status}");
            return length;
        }
        catch (RedisConnectionException ex)
        {
            Console.Error.WriteLine($"Redis connection failed during STRLEN {key}: {ex.Message}");
            throw;
        }
        catch (RedisServerException ex) when (ex.Message.Contains("WRONGTYPE"))
        {
            Console.Error.WriteLine($"WRONGTYPE: Key '{key}' is not a string.");
            throw;
        }
        catch (RedisServerException ex)
        {
            Console.Error.WriteLine($"Redis server error during STRLEN {key}: {ex.Message}");
            throw;
        }
        catch (TimeoutException ex)
        {
            Console.Error.WriteLine($"Timeout during STRLEN {key}: {ex.Message}");
            throw;
        }
    }

    /// <summary>
    /// Compares byte length vs. character count for UTF-8 strings.
    /// </summary>
    public static async Task CompareByteVsCharLengthAsync(string key)
    {
        var db = await GetDbAsync();

        try
        {
            string value = await db.StringGetAsync(key);

            if (value == null)
            {
                Console.WriteLine($"Key '{key}' does not exist.");
                return;
            }

            long byteLength = await db.StringLengthAsync(key);
            int charCount = value.Length;
            int utf8ByteCount = System.Text.Encoding.UTF8.GetByteCount(value);

            Console.WriteLine($"String: \"{value}\"");
            Console.WriteLine($"  Character count (C#): {charCount}");
            Console.WriteLine($"  STRLEN (Redis): {byteLength}");
            Console.WriteLine($"  UTF-8 byte count: {utf8ByteCount}");

            if (byteLength != charCount)
            {
                Console.WriteLine($"  ⚠ Difference: {byteLength - charCount} bytes — multi-byte characters present");
            }
        }
        catch (RedisConnectionException ex)
        {
            Console.Error.WriteLine($"Redis connection failed: {ex.Message}");
            throw;
        }
        catch (RedisServerException ex)
        {
            Console.Error.WriteLine($"Redis server error: {ex.Message}");
            throw;
        }
        catch (TimeoutException ex)
        {
            Console.Error.WriteLine($"Timeout: {ex.Message}");
            throw;
        }
    }

    /// <summary>
    /// Uses STRLEN to decide whether to read a value (avoids loading large strings).
    /// </summary>
    public static async Task ConditionalReadAsync(string key, long maxReadSize = 1024 * 1024)
    {
        var db = await GetDbAsync();

        try
        {
            long length = await db.StringLengthAsync(key);

            if (length == 0)
            {
                Console.WriteLine($"Key '{key}' is empty or does not exist.");
                return;
            }

            if (length > maxReadSize)
            {
                Console.WriteLine($"Key '{key}' is {length} bytes — exceeds max read size of {maxReadSize} bytes.");
                Console.WriteLine("Consider reading with GETRANGE instead.");
                return;
            }

            // Safe to read
            RedisValue value = await db.StringGetAsync(key);
            Console.WriteLine($"Key '{key}': {length} bytes, value: \"{value}\"");
        }
        catch (RedisConnectionException ex)
        {
            Console.Error.WriteLine($"Redis connection failed: {ex.Message}");
            throw;
        }
```

```csharp
/// <summary>
/// Complete string manipulation service combining APPEND, STRLEN, GETRANGE, SETRANGE.
/// Production-ready with error handling and logging.
/// </summary>
public sealed class StringManipulationService
{
    private readonly IDatabase _db;
    private const long MaxStringSize = 512 * 1024 * 1024; // 512 MB

    public StringManipulationService(IDatabase db)
    {
        _db = db ?? throw new ArgumentNullException(nameof(db));
    }

    /// <summary>
    /// APPEND with size limit enforcement.
    /// </summary>
    public async Task<(bool Success, long NewLength)> TryAppendAsync(
        string key, string value)
    {
        try
        {
            long currentLength = await _db.StringLengthAsync(key);
            long valueBytes = System.Text.Encoding.UTF8.GetByteCount(value);

            if (currentLength + valueBytes > MaxStringSize)
            {
                Console.Error.WriteLine($"APPEND would exceed 512 MB limit for '{key}'");
                return (false, currentLength);
            }

            long newLength = await _db.StringAppendAsync(key, value);
            return (true, newLength);
        }
        catch (RedisConnectionException ex)
        {
            Console.Error.WriteLine($"Redis connection error in TryAppend: {ex.Message}");
            throw;
        }
        catch (RedisServerException ex)
        {
            Console.Error.WriteLine($"Redis server error in TryAppend: {ex.Message}");
            throw;
        }
        catch (TimeoutException ex)
        {
            Console.Error.WriteLine($"Timeout in TryAppend: {ex.Message}");
            throw;
        }
    }

    /// <summary>
    /// GETRANGE with UTF-8 safety check.
    /// </summary>
    public async Task<string> GetRangeSafeAsync(string key, long start, long end)
    {
        try
        {
            RedisValue result = await _db.StringGetRangeAsync(key, start, end);

            if (result.IsNull)
                return null;

            // Check if the result is valid UTF-8
            byte[] bytes = (byte[])result;
            try
            {
                string text = System.Text.Encoding.UTF8.GetString(bytes);
                // Verify round-trip
                byte[] reencoded = System.Text.Encoding.UTF8.GetBytes(text);
                if (bytes.Length != reencoded.Length || !bytes.SequenceEqual(reencoded))
                {
                    Console.WriteLine($"Warning: GETRANGE returned invalid UTF-8 at bytes [{start}..{end}]");
                }
                return text;
            }
            catch (DecoderFallbackException)
            {
                Console.WriteLine($"GETRANGE returned invalid UTF-8 at bytes [{start}..{end}]");
                return null;
            }
        }
        catch (RedisConnectionException ex)
        {
            Console.Error.WriteLine($"Redis connection error in GetRangeSafe: {ex.Message}");
            throw;
        }
        catch (RedisServerException ex)
        {
            Console.Error.WriteLine($"Redis server error in GetRangeSafe: {ex.Message}");
            throw;
        }
        catch (TimeoutException ex)
        {
            Console.Error.WriteLine($"Timeout in GetRangeSafe: {ex.Message}");
            throw;
        }
    }

    /// <summary>
    /// SETRANGE with size limit enforcement.
    /// </summary>
    public async Task<(bool Success, long NewLength)> TrySetRangeAsync(
        string key, long offset, string value)
    {
        try
        {
            long currentLength = await _db.StringLengthAsync(key);
            long valueBytes = System.Text.Encoding.UTF8.GetByteCount(value);
            long requiredLength = Math.Max(currentLength, offset + valueBytes);

            if (requiredLength > MaxStringSize)
            {
                Console.Error.WriteLine($"SETRANGE would exceed 512 MB limit for '{key}'");
                return (false, currentLength);
            }

            long newLength = await _db.StringSetRangeAsync(key, offset, value);
            return (true, newLength);
        }
        catch (RedisConnectionException ex)
        {
            Console.Error.WriteLine($"Redis connection error in TrySetRange: {ex.Message}");
            throw;
        }
        catch (RedisServerException ex)
        {
            Console.Error.WriteLine($"Redis server error in TrySetRange: {ex.Message}");
            throw;
        }
        catch (TimeoutException ex)
        {
            Console.Error.WriteLine($"Timeout in TrySetRange: {ex.Message}");
            throw;
        }
    }

    /// <summary>
    /// Efficient buffer — uses STRLEN to check before reading.
    /// </summary>
    public async Task<long> GetLengthAsync(string key)
    {
        try
        {
            return await _db.StringLengthAsync(key);
        }
        catch (RedisConnectionException ex)
        {
            Console.Error.WriteLine($"Redis connection error in GetLength: {ex.Message}");
            throw;
        }
        catch (RedisServerException ex)
        {
            Console.Error.WriteLine($"Redis server error in GetLength: {ex.Message}");
            throw;
        }
        catch (TimeoutException ex)
        {
            Console.Error.WriteLine($"Timeout in GetLength: {ex.Message}");
            throw;
        }
    }

    /// <summary>
    /// Builds a string incrementally with APPEND, monitoring size.
    /// </summary>
    public async Task<long> BuildStringAsync(string key, IEnumerable<string> parts)
    {
        long totalLength = 0;

        foreach (var part in parts)
        {
            var (success, newLength) = await TryAppendAsync(key, part);
            if (!success)
            {
                Console.WriteLine($"Stopped building '{key}' at {totalLength} bytes (512 MB limit)");
                break;
            }
            totalLength = newLength;
        }

        Console.WriteLine($"Built string '{key}': {totalLength} bytes from {parts.Count()} parts");
        return totalLength;
    }
```

```csharp
            Console.Error.WriteLine($"Redis server error appending to '{fileKey}': {ex.Message}");
            throw;
        }
        catch (TimeoutException ex)
        {
            Console.Error.WriteLine($"Timeout appending to '{fileKey}': {ex.Message}");
            throw;
        }
    }

    /// <summary>
    /// Gets the file size in bytes.
    /// </summary>
    public async Task<long> GetSizeAsync(string fileKey)
    {
        return await _db.StringLengthAsync(fileKey);
    }

    /// <summary>
    /// Truncates the file to a specified size by setting it to a GETRANGE slice.
    /// </summary>
    public async Task TruncateAsync(string fileKey, long newSize)
    {
        try
        {
            long currentSize = await _db.StringLengthAsync(fileKey);
            if (newSize >= currentSize)
                return; // No truncation needed

            RedisValue truncated = await _db.StringGetRangeAsync(
                fileKey, 0, newSize - 1);

            await _db.StringSetAsync(fileKey, truncated);
            Console.WriteLine($"Truncated '{fileKey}' from {currentSize} to {newSize} bytes");
        }
        catch (RedisConnectionException ex)
        {
            Console.Error.WriteLine($"Redis connection error truncating '{fileKey}': {ex.Message}");
            throw;
        }
        catch (RedisServerException ex)
        {
            Console.Error.WriteLine($"Redis server error truncating '{fileKey}': {ex.Message}");
            throw;
        }
        catch (TimeoutException ex)
        {
            Console.Error.WriteLine($"Timeout truncating '{fileKey}': {ex.Message}");
            throw;
        }
    }
}
```

## Section 6 — Performance Considerations

### O-Complexity

| Command | O-complexity | Details |
|---------|-------------|---------|
| APPEND | O(1) amortized | Redis uses SDS (Simple Dynamic String) with pre-allocation. Appending within pre-allocated capacity is O(1). Reallocation happens only when capacity is exceeded. |
| STRLEN | O(1) | Redis stores the string length in the SDS header — no traversal needed. |
| GETRANGE | O(N) | N is the length of the returned substring. Redis copies the requested bytes from the source string. |
| SETRANGE | O(1) amortized | Writing within existing capacity is O(1). Growing the string requires reallocation. Padding with null bytes is O(N) for the padding region. |

### Memory Allocation Behavior

```csharp
public static class MemoryBehavior
{
    /// <summary>
    /// SDS (Simple Dynamic String) pre-allocation strategy:
    /// When a string grows, Redis allocates capacity = min(max(next_power_of_two(length)), 1MB)
    /// This means APPEND has amortized O(1) time.
    /// </summary>
    public static async Task DemonstratePreallocationAsync(IDatabase db)
    {
        string key = "demo:sds";

        // Start with a small string
        await db.StringSetAsync(key, "x");

        // First append — may trigger reallocation
        long len1 = await db.StringAppendAsync(key, new string('x', 100));
        Console.WriteLine($"After append 100 bytes: {len1}");

        // Subsequent appends within pre-allocated capacity are very fast
        var sw = System.Diagnostics.Stopwatch.StartNew();
        for (int i = 0; i < 1000; i++)
        {
            await db.StringAppendAsync(key, "x");
        }
        sw.Stop();
        Console.WriteLine($"1000 appends: {sw.Elapsed.TotalMilliseconds:F3} ms — very fast due to pre-allocation");

        await db.KeyDeleteAsync(key);
    }
}
```

### Network Efficiency

For string manipulation commands:

| Pattern | Round trips | Data transferred | Use case |
|---------|-------------|-----------------|----------|
| GET full + modify + SET | 2 | Entire value | Small values (< 1 KB) |
| GETRANGE + SETRANGE | 2 | Partial value | Large values, known offsets |

```csharp
        return true;

        // Process the value
```

### Large String Handling

```csharp
/// <summary>
/// Best practices for handling large Redis strings (> 10 MB).
/// </summary>
public static class LargeStringGuidelines
{
    // For values > 10 MB, prefer GETRANGE/SETRANGE instead of GET/SET
    // This avoids allocating large byte arrays and reduces memory pressure

    /// <summary>
    /// For very large values, process in chunks.
    /// </summary>
    public static async Task ProcessLargeInChunksAsync(
        IDatabase db, string key, int chunkSize = 1024 * 1024)
    {
        long offset = 0;
        int totalProcessed = 0;

        while (true)
        {
            RedisValue chunk = await db.StringGetRangeAsync(
                key, offset, offset + chunkSize - 1);

            if (chunk.IsNull || chunk.Length() == 0)
                break;

            byte[] data = (byte[])chunk;
            await ProcessChunkAsync(data, offset);

            totalProcessed += data.Length;
            offset += data.Length;

            if (data.Length < chunkSize)
                break; // Last chunk
        }

        Console.WriteLine($"Processed {totalProcessed} bytes in chunks");
    }

    private static Task ProcessChunkAsync(byte[] data, long offset)
    {
        Console.WriteLine($"Processing chunk at offset {offset}: {data.Length} bytes");
        return Task.CompletedTask;
    }
}
```

## Section 7 — Production Considerations

### 512 MB String Limit

```csharp
/// <summary>
/// Redis strings have a maximum size of 512 MB.
/// APPEND and SETRANGE will fail if they would exceed this limit.
/// </summary>
public static class SizeLimitEnforcement
{
    public const long MaxStringBytes = 512 * 1024 * 1024; // 536,870,912 bytes

    /// <summary>
    /// Checks if an APPEND operation would exceed the string size limit.
    /// </summary>
    public static async Task<bool> CanAppendAsync(IDatabase db, string key, string value)
    {
        long currentLength = await db.StringLengthAsync(key);
        int appendBytes = System.Text.Encoding.UTF8.GetByteCount(value);
        return currentLength + appendBytes <= MaxStringBytes;
    }

    /// <summary>
    /// Checks if a SETRANGE operation would exceed the string size limit.
    /// </summary>
    public static async Task<bool> CanSetRangeAsync(IDatabase db, string key, long offset, string value)
    {
        long currentLength = await db.StringLengthAsync(key);
        int valueBytes = System.Text.Encoding.UTF8.GetByteCount(value);
        long requiredSize = Math.Max(currentLength, offset + valueBytes);
        return requiredSize <= MaxStringBytes;
    }

    /// <summary>
    /// For data that exceeds 512 MB, consider alternatives:
    /// - Split into multiple keys (string sharding)
    /// - Use Streams for log data
    /// - Use a file/blob store for large blobs
    /// </summary>
    public static List<string> GetShardKeys(string baseKey, long totalSize)
    {
        const long maxShardSize = 500 * 1024 * 1024; // 500 MB per shard (buffer)
        int shardCount = (int)Math.Ceiling((double)totalSize / maxShardSize);
        var keys = new List<string>(shardCount);

        for (int i = 0; i < shardCount; i++)
        {
            keys.Add($"{baseKey}:shard:{i}");
        }

        return keys;
    }
}
```

```csharp
        Console.WriteLine($"Length: {length}"); // 15

        RedisValue value = await db.StringGetAsync("demo:nullpad");
        Console.WriteLine($"As string: \"{value}\""); // May show only "" due to null bytes

        byte[] bytes = (byte[])value;
        Console.WriteLine($"Bytes: {BitConverter.ToString(bytes)}");

        // GETRANGE can extract the meaningful part
        RedisValue meaningful = await db.StringGetRangeAsync("demo:nullpad", 10, 14);
        Console.WriteLine($"GETRANGE meaningful part: \"{meaningful}\""); // "hello"

        await db.KeyDeleteAsync("demo:nullpad");
    }

    /// <summary>
    /// Avoid creating null bytes by initializing strings with proper sizing.
    /// </summary>
    public static async Task SafeSetRangeAsync(IDatabase db, string key, long offset, string value)
    {
        long currentLength = await db.StringLengthAsync(key);

        if (offset > currentLength)
        {
            // Initialize with spaces instead of relying on null byte padding
            string padding = new string(' ', (int)(offset - currentLength));
            await db.StringAppendAsync(key, padding);
        }

        await db.StringSetRangeAsync(key, offset, value);
    }
}
```

### UTF-8 Encoding Awareness

```csharp
/// <summary>
/// GETRANGE and SETRANGE operate on BYTES, not characters.
/// For UTF-8 strings, this can split multi-byte characters.
/// </summary>
public static class Utf8AwareOperations
{
    /// <summary>
    /// Checks if a byte range is safe for UTF-8 (does not split multi-byte chars).
    /// </summary>
    public static bool IsSafeUtf8Range(string text, long startByte, long endByte)
    {
        byte[] bytes = System.Text.Encoding.UTF8.GetBytes(text);

        if (startByte >= bytes.Length || endByte >= bytes.Length)
            return false;

        // Check that startByte is at a character boundary
        if (startByte > 0)
        {
            // Scan backwards to find the start of the current character
            int i = (int)startByte;
            while (i > 0 && (bytes[i] & 0xC0) == 0x80)
                i--;
            if (i != startByte)
                return false; // startByte is in the middle of a multi-byte character
        }

        // Check that endByte is at a character boundary (inclusive)
        if (endByte < bytes.Length - 1)
        {
            int i = (int)endByte + 1;
            while (i < bytes.Length && (bytes[i] & 0xC0) == 0x80)
                i++;
            if (i != endByte + 1)
                return false; // endByte is in the middle of a multi-byte character
        }

        return true;
    }

    /// <summary>
    /// Safe byte-range extraction for UTF-8 strings.
    /// Adjusts the range to avoid splitting multi-byte characters.
    /// </summary>
    public static (long Start, long End) GetSafeUtf8Range(
        string text, long desiredStart, long desiredEnd)
    {
        byte[] bytes = System.Text.Encoding.UTF8.GetBytes(text);

        long safeStart = desiredStart;
        long safeEnd = desiredEnd;

        // Adjust start forward if it's in the middle of a multi-byte char
        if (safeStart > 0 && safeStart < bytes.Length)
        {
            int i = (int)safeStart;
            while (i > 0 && (bytes[i] & 0xC0) == 0x80)
                i--;
            safeStart = i;
        }

        // Adjust end backward if it's in the middle of a multi-byte char
        if (safeEnd < bytes.Length - 1)
        {
            int i = (int)safeEnd;
            while (i > 0 && (bytes[i] & 0xC0) == 0x80)
                i--;
            if (i == safeEnd) // We're at the start of a char
                safeEnd = safeEnd; // OK
            else
                safeEnd = i - 1; // Move back to end of previous char
        }

        return (safeStart, safeEnd);
    }
}
```

### GETRANGE/SETRANGE Byte Index Confusion

```csharp
public static async Task ByteIndexConfusionAsync(IDatabase db)
{
    // GETRANGE and SETRANGE use BYTE indices, not character indices
    // This is a common source of bugs when working with non-ASCII text

    await db.StringSetAsync("utf8demo", "café");

    // Getting bytes 0-2 returns "caf" (3 bytes, 3 characters — safe for ASCII range)
    RedisValue safe = await db.StringGetRangeAsync("utf8demo", 0, 2);
    Console.WriteLine($"GETRANGE 0-2: \"{safe}\""); // "caf"

    // Getting bytes 0-3 splits 'é' (2 bytes: C3 A9)
    RedisValue broken = await db.StringGetRangeAsync("utf8demo", 0, 3);
    Console.WriteLine($"GETRANGE 0-3: \"{broken}\""); // "caf\xC3" — invalid UTF-8

    // Getting bytes 0-4 gets the complete character
    RedisValue complete = await db.StringGetRangeAsync("utf8demo", 0, 4);
    Console.WriteLine($"GETRANGE 0-4: \"{complete}\""); // "café"

    // STRLEN returns byte length, not character count
    long byteLen = await db.StringLengthAsync("utf8demo");
    Console.WriteLine($"STRLEN: {byteLen}"); // 5 (not 4)
}
```
