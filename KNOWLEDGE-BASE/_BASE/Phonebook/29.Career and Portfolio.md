## 29.01 — Source Generators

**What it is**
Compile-time code generation. The C# compiler runs your generator as part of compilation, emitting additional C# source files. No runtime reflection overhead.

**Use cases**
- JSON serialization without reflection (`System.Text.Json` source gen)
- Dependency injection registration without reflection
- Logging source generation (no boxing, no string interpolation at runtime)
- Regex compilation (no runtime regex compilation)
- AutoMapper-like mapping without reflection

**System.Text.Json source generation**
```csharp
// Define serializable types in a context
[JsonSerializable(typeof(Order))]
[JsonSerializable(typeof(List<Order>))]
[JsonSerializable(typeof(CreateOrderRequest))]
[JsonSourceGenerationOptions(
    PropertyNamingPolicy = JsonKnownNamingPolicy.CamelCase,
    WriteIndented = false)]
public partial class AppJsonContext : JsonSerializerContext { }

// Registration in ASP.NET Core — serializer uses generated code, not reflection
builder.Services.ConfigureHttpJsonOptions(options =>
{
    options.SerializerOptions.TypeInfoResolverChain.Insert(0, AppJsonContext.Default);
});

// Serialization — uses compiled code
var json = JsonSerializer.Serialize(order, AppJsonContext.Default.Order);
var order = JsonSerializer.Deserialize(json, AppJsonContext.Default.Order);
```

**Logging source generation (no boxing, no allocation)**
```csharp
public static partial class Log
{
    [LoggerMessage(Level = LogLevel.Information, 
                   Message = "Order {OrderId} created for customer {CustomerId}")]
    public static partial void OrderCreated(ILogger logger, int orderId, int customerId);
    
    [LoggerMessage(Level = LogLevel.Warning,
                   Message = "Retry attempt {Attempt} for operation {Operation}")]
    public static partial void RetryAttempt(ILogger logger, int attempt, string operation);
    
    [LoggerMessage(Level = LogLevel.Error,
                   Message = "Failed to process order {OrderId}")]
    public static partial void OrderProcessingFailed(ILogger logger, Exception ex, int orderId);
}

// Usage — no boxing, no string allocation if log level is disabled
Log.OrderCreated(logger, order.Id, order.CustomerId);
```

**Regex source generation**
```csharp
public partial class EmailValidator
{
    [GeneratedRegex(@"^[^@\s]+@[^@\s]+\.[^@\s]+$", RegexOptions.IgnoreCase)]
    private static partial Regex EmailRegex();
    
    public bool IsValid(string email) => EmailRegex().IsMatch(email);
}
// The regex is compiled at build time — no runtime Regex.Compile() cost
```

## 29.02 — Primary Constructors (C# 12)

**What it is**
Parameters declared in the class/struct declaration, available throughout the entire class body. Reduces boilerplate for dependency injection.

```csharp
// Before (C# 11 and earlier)
public class OrderService
{
    private readonly IOrderRepository _repository;
    private readonly IEmailService _emailService;
    private readonly ILogger<OrderService> _logger;
    
    public OrderService(
        IOrderRepository repository,
        IEmailService emailService,
        ILogger<OrderService> logger)
    {
        _repository = repository;
        _emailService = emailService;
        _logger = logger;
    }
    
    public async Task<Order?> GetAsync(int id)
    {
        _logger.LogInformation("Getting order {OrderId}", id);
        return await _repository.GetByIdAsync(id);
    }
}

// After (C# 12 primary constructors)
public class OrderService(
    IOrderRepository repository,
    IEmailService emailService,
    ILogger<OrderService> logger)
{
    public async Task<Order?> GetAsync(int id)
    {
        logger.LogInformation("Getting order {OrderId}", id);  // Direct parameter access
        return await repository.GetByIdAsync(id);
    }
    
    public async Task SendConfirmationAsync(int orderId)
    {
        var order = await repository.GetByIdAsync(orderId);
        await emailService.SendAsync(order!.CustomerEmail, "Order confirmed", "...");
    }
}
```

**Caution with primary constructors**
Primary constructor parameters are captured if used in methods. If you use them in a lambda, they will be heap-allocated (captured in a closure). If you need the field explicitly (for encapsulation, reflection, or to document intent), still declare a private field.

## 29.03 — Discriminated Unions with OneOf

**The problem**
C# does not have native discriminated unions. Methods that return "either a value or an error" often resort to exceptions, tuples, or nullable types — all with drawbacks.

**OneOf library approach**
```csharp
// NuGet: OneOf
public class OrderService
{
    public async Task<OneOf<Order, NotFound, ValidationError>> GetOrderAsync(int id)
    {
        if (id <= 0)
            return new ValidationError("Order ID must be positive");
        
        var order = await _repository.GetByIdAsync(id);
        if (order is null)
            return new NotFound();
        
        return order;
    }
}

// Usage — exhaustive pattern matching, no runtime exceptions
var result = await _orderService.GetOrderAsync(orderId);

return result.Match(
    order => Results.Ok(order.ToDto()),
    notFound => Results.NotFound(),
    error => Results.BadRequest(error.Message));
```

**Result<T, TError> pattern (roll your own)**
```csharp
public readonly struct Result<T, TError>
{
    private readonly T? _value;
    private readonly TError? _error;
    public bool IsSuccess { get; }
    
    private Result(T value) { _value = value; IsSuccess = true; _error = default; }
    private Result(TError error) { _error = error; IsSuccess = false; _value = default; }
    
    public static Result<T, TError> Ok(T value) => new(value);
    public static Result<T, TError> Fail(TError error) => new(error);
    
    public TResult Match<TResult>(Func<T, TResult> onSuccess, Func<TError, TResult> onFailure)
        => IsSuccess ? onSuccess(_value!) : onFailure(_error!);
    
    public static implicit operator Result<T, TError>(T value) => Ok(value);
    public static implicit operator Result<T, TError>(TError error) => Fail(error);
}

// Usage
public async Task<Result<OrderDto, DomainError>> CreateOrderAsync(CreateOrderCommand cmd)
{
    if (cmd.Items.Count == 0)
        return new DomainError("Order must have at least one item");
    
    var order = Order.Create(cmd.CustomerId);
    await _repository.SaveAsync(order);
    return order.ToDto();
}

// Caller
var result = await _orderService.CreateOrderAsync(command);
return result.Match(
    dto => Results.Created($"/api/orders/{dto.Id}", dto),
    error => Results.UnprocessableEntity(error.Message));
```

## 29.04 — Minimal APIs — Deep Understanding

**What it is**
ASP.NET Core minimal APIs provide a concise, low-overhead approach to building HTTP endpoints without the ceremony of Controllers.

**Complete minimal API**
```csharp
// Program.cs
var builder = WebApplication.CreateBuilder(args);
builder.Services.AddEndpointsApiExplorer();
builder.Services.AddSwaggerGen();

var app = builder.Build();

// Map endpoints using extension methods for organization
app.MapOrderEndpoints();
app.MapProductEndpoints();

app.Run();

// OrderEndpoints.cs
public static class OrderEndpoints
{
    public static IEndpointRouteBuilder MapOrderEndpoints(this IEndpointRouteBuilder app)
    {
        var group = app.MapGroup("/api/orders")
            .WithTags("Orders")
            .RequireAuthorization();
        
        group.MapGet("", GetOrders)
            .WithName("GetOrders")
            .Produces<PagedResult<OrderSummaryDto>>()
            .Produces(401);
        
        group.MapGet("{id:int}", GetOrderById)
            .WithName("GetOrderById")
            .Produces<OrderDetailDto>()
            .Produces(404)
            .Produces(401);
        
        group.MapPost("", CreateOrder)
            .WithName("CreateOrder")
            .Produces<CreateOrderResponse>(201)
            .ProducesValidationProblem()
            .Produces(401);
        
        group.MapDelete("{id:int}", CancelOrder)
            .WithName("CancelOrder")
            .RequireAuthorization("OrderOwner")
            .Produces(204)
            .Produces(404)
            .Produces(403);
        
        return app;
    }
    
    private static async Task<IResult> GetOrders(
        [FromServices] ISender mediator,
        [FromQuery] int page = 1,
        [FromQuery] int pageSize = 20,
        [FromQuery] string? status = null,
        CancellationToken ct = default)
    {
        var query = new GetOrdersQuery(page, pageSize, status);
        var result = await mediator.Send(query, ct);
        return Results.Ok(result);
    }
    
    private static async Task<IResult> GetOrderById(
        int id,
        [FromServices] ISender mediator,
        CancellationToken ct)
    {
        var result = await mediator.Send(new GetOrderByIdQuery(id), ct);
        return result is null ? Results.NotFound() : Results.Ok(result);
    }
    
    private static async Task<IResult> CreateOrder(
        [FromBody] CreateOrderRequest request,
        [FromServices] ISender mediator,
        [FromServices] IValidator<CreateOrderRequest> validator,
        CancellationToken ct)
    {
        var validation = await validator.ValidateAsync(request, ct);
        if (!validation.IsValid)
            return Results.ValidationProblem(validation.ToDictionary());
        
        var command = new CreateOrderCommand(request.CustomerId, request.Items);
        var result = await mediator.Send(command, ct);
        return Results.Created($"/api/orders/{result.OrderId}", result);
    }
    
    private static async Task<IResult> CancelOrder(
        int id,
        [FromServices] ISender mediator,
        CancellationToken ct)
    {
        await mediator.Send(new CancelOrderCommand(id), ct);
        return Results.NoContent();
    }
}
```

**TypedResults vs Results**
```csharp
// Results — returns IResult, no compile-time type info for OpenAPI
static IResult GetOrder(int id) => Results.Ok(new OrderDto());

// TypedResults — returns typed result, OpenAPI knows the type automatically
static Ok<OrderDto> GetOrder(int id) => TypedResults.Ok(new OrderDto());

// Endpoint with multiple typed results:
static Results<Ok<OrderDto>, NotFound, ForbidHttpResult> GetOrder(int id)
{
    var order = GetFromDb(id);
    if (order is null) return TypedResults.NotFound();
    if (!CanAccess(order)) return TypedResults.Forbid();
    return TypedResults.Ok(order.ToDto());
}
```

**Filters (equivalent of action filters)**
```csharp
// Endpoint filter for validation
app.MapPost("/api/orders", CreateOrder)
    .AddEndpointFilter<ValidationFilter<CreateOrderRequest>>();

public class ValidationFilter<T> : IEndpointFilter where T : class
{
    private readonly IValidator<T> _validator;
    public ValidationFilter(IValidator<T> validator) => _validator = validator;
    
    public async ValueTask<object?> InvokeAsync(
        EndpointFilterInvocationContext ctx,
        EndpointFilterDelegate next)
    {
        var request = ctx.GetArgument<T>(0);
        var result = await _validator.ValidateAsync(request);
        
        if (!result.IsValid)
            return Results.ValidationProblem(result.ToDictionary());
        
        return await next(ctx);
    }
}
```

## 29.05 — Channels and Pipelines

**Real-world pipeline: Image processing**
```csharp
public class ImageProcessingPipeline
{
    public async Task RunAsync(string[] imagePaths, CancellationToken ct)
    {
        // Stage 1 → Stage 2 → Stage 3
        var loadChannel = Channel.CreateBounded<(string path, byte[] data)>(50);
        var processChannel = Channel.CreateBounded<(string path, byte[] processed)>(50);
        
        await Task.WhenAll(
            LoadImagesAsync(imagePaths, loadChannel.Writer, ct),
            ProcessImagesAsync(loadChannel.Reader, processChannel.Writer, ct),
            SaveImagesAsync(processChannel.Reader, ct)
        );
    }
    
    private async Task LoadImagesAsync(
        string[] paths,
        ChannelWriter<(string, byte[])> writer,
        CancellationToken ct)
    {
        try
        {
            foreach (var path in paths)
            {
                ct.ThrowIfCancellationRequested();
                var data = await File.ReadAllBytesAsync(path, ct);
                await writer.WriteAsync((path, data), ct);
            }
        }
        finally
        {
            writer.Complete();
        }
    }
    
    private async Task ProcessImagesAsync(
        ChannelReader<(string, byte[])> reader,
        ChannelWriter<(string, byte[])> writer,
        CancellationToken ct)
    {
        try
        {
            await foreach (var (path, data) in reader.ReadAllAsync(ct))
            {
                var processed = await ResizeAndOptimizeAsync(data, ct);
                await writer.WriteAsync((path, processed), ct);
            }
        }
        finally
        {
            writer.Complete();
        }
    }
    
    private async Task SaveImagesAsync(
        ChannelReader<(string, byte[])> reader,
        CancellationToken ct)
    {
        await foreach (var (path, data) in reader.ReadAllAsync(ct))
        {
            var outputPath = Path.ChangeExtension(path, ".processed.webp");
            await File.WriteAllBytesAsync(outputPath, data, ct);
        }
    }
}
```
