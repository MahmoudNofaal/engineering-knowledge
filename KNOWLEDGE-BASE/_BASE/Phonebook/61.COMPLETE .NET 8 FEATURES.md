## 61.01 — What's New and Important in .NET 8

**Primary Constructors (already covered) — also for non-record classes**

**Collection expressions**
```csharp
// New unified syntax for creating collections
int[] array = [1, 2, 3, 4, 5];
List<int> list = [1, 2, 3];
HashSet<int> set = [1, 2, 3];
Span<int> span = [1, 2, 3];

// Spread operator
int[] a = [1, 2, 3];
int[] b = [4, 5, 6];
int[] combined = [..a, ..b];  // [1, 2, 3, 4, 5, 6]
List<int> mixed = [0, ..a, ..b, 7];

// In method calls
void Process(IEnumerable<int> items) { }
Process([1, 2, 3]);  // Works for any collection type
```

**Time abstraction**
```csharp
// Before .NET 8: hard to test time-dependent code
// After .NET 8: inject TimeProvider

builder.Services.AddSingleton(TimeProvider.System);

// In service
public class OrderService
{
    private readonly TimeProvider _time;
    
    public OrderService(TimeProvider time) => _time = time;
    
    public Order CreateOrder(int customerId)
    {
        return new Order
        {
            CreatedAt = _time.GetUtcNow(),  // Testable!
            ExpiresAt = _time.GetUtcNow().AddHours(24)
        };
    }
}

// In tests: use FakeTimeProvider
var fakeTime = new FakeTimeProvider();
fakeTime.SetUtcNow(new DateTimeOffset(2026, 1, 1, 12, 0, 0, TimeSpan.Zero));

var sut = new OrderService(fakeTime);
var order = sut.CreateOrder(1);
Assert.Equal(new DateTime(2026, 1, 1, 12, 0, 0), order.CreatedAt);

fakeTime.Advance(TimeSpan.FromHours(25));  // Simulate time passing
```

**Metrics API improvements**
```csharp
// System.Diagnostics.Metrics — .NET 5+ but improved in .NET 8
var meter = new Meter("MyApp.Orders", "1.0.0");

// Counter
var orderCreatedCounter = meter.CreateCounter<int>(
    "orders.created",
    "orders",
    "Number of orders created");

orderCreatedCounter.Add(1,
    new KeyValuePair<string, object?>("status", "pending"),
    new KeyValuePair<string, object?>("region", "us-east"));

// Histogram (for latency)
var requestDuration = meter.CreateHistogram<double>(
    "http.request.duration",
    "ms",
    "HTTP request duration");

requestDuration.Record(stopwatch.Elapsed.TotalMilliseconds,
    new KeyValuePair<string, object?>("method", "GET"),
    new KeyValuePair<string, object?>("route", "/api/orders"));

// Observable counter (pulled value)
meter.CreateObservableCounter(
    "orders.active",
    () => _orderService.GetActiveCount(),
    "orders",
    "Number of currently active orders");
```

**SearchValues<T> (fast multi-char search)**
```csharp
// Optimized for searching for any character in a set
private static readonly SearchValues<char> InvalidChars =
    SearchValues.Create("<>\"'&/\\");

bool ContainsInvalidChar(ReadOnlySpan<char> input)
    => input.ContainsAny(InvalidChars);  // Vectorized, very fast

// String splitting with SearchValues
private static readonly SearchValues<char> Separators =
    SearchValues.Create(new[] { ',', ';', '|', '\t' });

var fields = line.AsSpan().IndexOfAny(Separators);
```

**Garbage Collector improvements**
```csharp
// Dynamic adaptation to application sizes (DATAS) — .NET 8 default
// GC now adjusts heap size based on application memory usage pattern
// Applications that need more memory get it; those that don't give it back to OS

// Server GC vs Workstation GC
// In appHost or runtimeconfig.json:
// "System.GC.Server": true   // Server GC: parallel, for server apps
// "System.GC.Server": false  // Workstation GC: for desktop/client apps

// Configure via environment (Docker/K8s):
// DOTNET_GCConserveMemory=5  // 0-9, higher = more memory-conservative (slower)
// DOTNET_GCHeapHardLimit=1073741824  // Hard limit in bytes (1 GB)
```

## 61.02 — C# 12 Features

**Collection expressions** (shown above)

**Inline arrays**
```csharp
// Fixed-size buffer on stack, indexed like array
[System.Runtime.CompilerServices.InlineArray(16)]
public struct Buffer16<T>
{
    private T _element;
}

// Usage: stack-allocated, fixed size, zero GC
var buffer = new Buffer16<int>();
buffer[0] = 1;
buffer[15] = 16;
```

**Using aliases for any type**
```csharp
// C# 12: alias any type, not just named types
using Point = (double X, double Y);
using Matrix = double[][];
using OrderId = int;

Point origin = (0, 0);
OrderId id = 42;
```
