## 5.01 — Array

**What it is**
A contiguous block of memory holding elements of the same type, accessed by index in O(1).

**Properties**
- **Access by index**: O(1)
- **Search (unsorted)**: O(n)
- **Search (sorted)**: O(log n) with binary search
- **Insert at end**: O(1) amortized (dynamic arrays like List<T>)
- **Insert at middle**: O(n) — must shift elements
- **Delete from middle**: O(n) — must shift elements
- **Memory**: Contiguous — excellent cache locality

**In .NET**
```csharp
int[] arr = new int[10];
int[] arr2 = { 1, 2, 3, 4, 5 };

var list = new List<int> { 1, 2, 3 };
list.Add(4);               // O(1) amortized
list.Insert(1, 99);        // O(n)
list.RemoveAt(2);          // O(n)
list.Contains(99);         // O(n)
list.BinarySearch(3);      // O(log n) - requires sorted

Array.Sort(arr);
Array.Reverse(arr);
Array.Copy(source, dest, length);

ReadOnlySpan<int> slice = arr.AsSpan(2, 5);  // No copy
```

**When to use**
- Random access by index
- Cache-friendly iteration
- When size is known upfront

**Gotchas**
- List<T> capacity doubles when exceeded — set capacity upfront: new List<T>(expectedCount)
- Multi-dimensional arrays (int[,]) vs jagged arrays (int[][]): jagged has better row-access cache behavior

## 5.02 — Linked List

**What it is**
A sequence of nodes where each node contains a value and a reference to the next node.

**Properties**
- **Access by index**: O(n)
- **Search**: O(n)
- **Insert at head/tail**: O(1)
- **Insert given node reference**: O(1)
- **Delete given node reference**: O(1)
- **Memory**: Non-contiguous — poor cache locality

**In .NET**
```csharp
var list = new LinkedList<int>();
var node1 = list.AddLast(1);
var node2 = list.AddLast(2);
list.AddBefore(node2, 99);
list.Remove(node1);

for (var node = list.First; node != null; node = node.Next)
    Console.WriteLine(node.Value);
```

**Classic interview patterns**
- Floyd's cycle detection (fast/slow pointers)
- Reverse a linked list
- Find the middle node
- Merge two sorted linked lists
- Remove Nth node from end

## 5.03 — Stack

**What it is**
A Last-In-First-Out (LIFO) data structure.

**Operations (all O(1))**
Push, Pop, Peek, IsEmpty.

**In .NET**
```csharp
var stack = new Stack<int>();
stack.Push(1);
stack.Push(2);
int top = stack.Peek();
int popped = stack.Pop();
bool has = stack.TryPop(out int result);
```

**Applications**
- Function call stack
- Undo/redo systems
- DFS traversal (iterative)
- Balanced parentheses
- Monotonic stack problems

**Monotonic stack (next greater element)**
```csharp
int[] NextGreaterElement(int[] nums)
{
    var result = Enumerable.Repeat(-1, nums.Length).ToArray();
    var stack = new Stack<int>(); // indices
    
    for (int i = 0; i < nums.Length; i++)
    {
        while (stack.Count > 0 && nums[stack.Peek()] < nums[i])
            result[stack.Pop()] = nums[i];
        stack.Push(i);
    }
    return result;
}
```

## 5.05 — Hash Map / Dictionary

**What it is**
A data structure mapping keys to values via hash function. O(1) average lookup, insert, delete.

**In .NET**
```csharp
var dict = new Dictionary<string, int>();
dict["Alice"] = 95;
dict.Add("Bob", 88);

bool has = dict.TryGetValue("Alice", out int score);
int val = dict.GetValueOrDefault("Charlie", 0);

foreach (var (key, value) in dict)
    Console.WriteLine($"{key}: {value}");

var set = new HashSet<int> { 1, 2, 3 };
set.Add(4);
set.Contains(2); // O(1)
set.UnionWith(other);
set.IntersectWith(other);

var concurrent = new ConcurrentDictionary<string, int>();
concurrent.AddOrUpdate("Alice", 95, (k, old) => old + 1);
int v = concurrent.GetOrAdd("Bob", k => ComputeDefault(k));
```

**Gotchas**
- Dictionary is not ordered (use SortedDictionary or OrderedDictionary from .NET 9)
- Custom key types need correct GetHashCode() and Equals()
- Mutable key objects break the dictionary if they change after insertion

## 5.06 — Priority Queue / Heap

**What it is**
A complete binary tree: min-heap has smallest element at root.

**Complexity**
- Min/Max access: O(1)
- Insert: O(log n)
- Remove min/max: O(log n)
- Build from array: O(n)

**In .NET**
```csharp
// PriorityQueue<TElement, TPriority> - min-heap (lowest value = highest priority)
var pq = new PriorityQueue<string, int>();
pq.Enqueue("Task A", 3);
pq.Enqueue("Task B", 1); // dequeued first
pq.Enqueue("Task C", 2);

string next = pq.Peek();  // "Task B"
pq.Dequeue();

// Max-heap: negate priorities
pq.Enqueue("Highest", -5);

bool has = pq.TryDequeue(out string element, out int priority);
```

**Applications**
- Dijkstra's algorithm
- Top-K problems
- Merge K sorted lists
- Median of data stream (two heaps)

**Median of data stream**
```csharp
public class MedianFinder
{
    private PriorityQueue<int, int> _lower = new(); // max-heap (negate priority)
    private PriorityQueue<int, int> _upper = new(); // min-heap

    public void AddNum(int num)
    {
        _lower.Enqueue(num, -num);
        
        if (_upper.Count > 0 && -_lower.Peek() > _upper.Peek())
        {
            var moved = _lower.Dequeue();
            _upper.Enqueue(moved, moved);
        }
        
        if (_lower.Count > _upper.Count + 1)
        {
            var top = _lower.Dequeue();
            _upper.Enqueue(top, top);
        }
        else if (_upper.Count > _lower.Count)
        {
            var top = _upper.Dequeue();
            _lower.Enqueue(top, -top);
        }
    }

    public double FindMedian()
    {
        if (_lower.Count == _upper.Count)
        {
            _lower.TryPeek(out int lo, out _);
            _upper.TryPeek(out int hi, out _);
            return (lo + hi) / 2.0;
        }
        return _lower.Peek();
    }
}
```

## 5.07 — Binary Tree Traversals

**In .NET**
```csharp
public class TreeNode
{
    public int Val;
    public TreeNode? Left, Right;
    public TreeNode(int val) => Val = val;
}

// Inorder: Left -> Root -> Right (sorted for BST)
void Inorder(TreeNode? node, List<int> result)
{
    if (node == null) return;
    Inorder(node.Left, result);
    result.Add(node.Val);
    Inorder(node.Right, result);
}

// Level order (BFS)
IList<IList<int>> LevelOrder(TreeNode? root)
{
    var result = new List<IList<int>>();
    if (root == null) return result;
    var queue = new Queue<TreeNode>();
    queue.Enqueue(root);
    
    while (queue.Count > 0)
    {
        int size = queue.Count;
        var level = new List<int>();
        for (int i = 0; i < size; i++)
        {
            var node = queue.Dequeue();
            level.Add(node.Val);
            if (node.Left != null) queue.Enqueue(node.Left);
            if (node.Right != null) queue.Enqueue(node.Right);
        }
        result.Add(level);
    }
    return result;
}

// Max depth
int MaxDepth(TreeNode? root) =>
    root == null ? 0 : 1 + Math.Max(MaxDepth(root.Left), MaxDepth(root.Right));

// Lowest Common Ancestor
TreeNode? LCA(TreeNode? root, TreeNode p, TreeNode q)
{
    if (root == null || root == p || root == q) return root;
    var left = LCA(root.Left, p, q);
    var right = LCA(root.Right, p, q);
    if (left != null && right != null) return root;
    return left ?? right;
}
```

## 5.08 — Graph

**Representations**
```csharp
// Adjacency list
var graph = new Dictionary<int, List<int>>();
graph[0] = new List<int> { 1, 2 };

// Adjacency matrix
bool[,] matrix = new bool[n, n];
matrix[0, 1] = true;
```

**BFS**
```csharp
int[] BFS(List<int>[] adj, int start, int n)
{
    var dist = Enumerable.Repeat(-1, n).ToArray();
    dist[start] = 0;
    var queue = new Queue<int>();
    queue.Enqueue(start);
    
    while (queue.Count > 0)
    {
        int node = queue.Dequeue();
        foreach (int neighbor in adj[node])
        {
            if (dist[neighbor] == -1)
            {
                dist[neighbor] = dist[node] + 1;
                queue.Enqueue(neighbor);
            }
        }
    }
    return dist;
}
```

**DFS**
```csharp
void DFS(List<int>[] adj, int node, bool[] visited)
{
    visited[node] = true;
    foreach (int neighbor in adj[node])
        if (!visited[neighbor])
            DFS(adj, neighbor, visited);
}
```

**Topological Sort (Kahn's)**
```csharp
int[] TopologicalSort(int n, int[][] edges)
{
    var inDegree = new int[n];
    var adj = Enumerable.Range(0, n).Select(_ => new List<int>()).ToArray();
    
    foreach (var e in edges)
    {
        adj[e[0]].Add(e[1]);
        inDegree[e[1]]++;
    }
    
    var queue = new Queue<int>();
    for (int i = 0; i < n; i++)
        if (inDegree[i] == 0) queue.Enqueue(i);
    
    var result = new List<int>();
    while (queue.Count > 0)
    {
        int node = queue.Dequeue();
        result.Add(node);
        foreach (int nb in adj[node])
            if (--inDegree[nb] == 0) queue.Enqueue(nb);
    }
    return result.Count == n ? result.ToArray() : Array.Empty<int>();
}
```

**Union-Find**
```csharp
public class UnionFind
{
    private int[] _parent, _rank;
    
    public UnionFind(int n)
    {
        _parent = Enumerable.Range(0, n).ToArray();
        _rank = new int[n];
    }
    
    public int Find(int x)
    {
        if (_parent[x] != x) _parent[x] = Find(_parent[x]);
        return _parent[x];
    }
    
    public bool Union(int x, int y)
    {
        int px = Find(x), py = Find(y);
        if (px == py) return false;
        if (_rank[px] < _rank[py]) (px, py) = (py, px);
        _parent[py] = px;
        if (_rank[px] == _rank[py]) _rank[px]++;
        return true;
    }
}
```

**Dijkstra**
```csharp
int[] Dijkstra(List<(int node, int weight)>[] adj, int source, int n)
{
    var dist = Enumerable.Repeat(int.MaxValue, n).ToArray();
    dist[source] = 0;
    var pq = new PriorityQueue<int, int>();
    pq.Enqueue(source, 0);
    
    while (pq.Count > 0)
    {
        pq.TryDequeue(out int node, out int d);
        if (d > dist[node]) continue;
        
        foreach (var (neighbor, weight) in adj[node])
        {
            int newDist = dist[node] + weight;
            if (newDist < dist[neighbor])
            {
                dist[neighbor] = newDist;
                pq.Enqueue(neighbor, newDist);
            }
        }
    }
    return dist;
}
```
