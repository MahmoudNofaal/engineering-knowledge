## 22.01 — LLM Fundamentals

**Tokens**
The unit of text an LLM processes. A token is roughly 4 characters or 0.75 words in English. "Hello world" ≈ 2 tokens. LLMs have a context window (max tokens) and are billed per token.

**Temperature**
Controls randomness of output. Range: 0 to 2.
- 0: Deterministic — always picks the highest probability token
- 0.7: Balanced — some creativity, still coherent
- 1.5: Creative/random — less predictable

Use low temperature for: structured outputs, factual Q&A, code generation.
Use high temperature for: creative writing, brainstorming.

**Top-p (nucleus sampling)**
Only considers tokens whose cumulative probability exceeds p. Top-p=0.9 means consider only tokens that together account for 90% of the probability. Reduces incoherent outputs.

**Context window**
Maximum number of tokens the model can process in one call (input + output combined). GPT-4o: 128K tokens. Claude 3.5 Sonnet: 200K tokens. Gemini 1.5: 1M tokens.

**Model selection by task**
| Task | Model choice | Reason |
|---|---|---|
| Simple classification | Small (Haiku, GPT-4o-mini) | Fast, cheap, sufficient |
| Complex reasoning | Large (Opus, GPT-4o) | Better accuracy |
| Code generation | Sonnet, GPT-4o | Specialized capability |
| Embeddings | text-embedding-3-small | Purpose-built for vectors |
| Real-time response | Small models | Latency requirement |

## 22.03 — RAG (Retrieval-Augmented Generation)

**What it is**
Enhancing LLM responses with relevant information retrieved from your own knowledge base. Solves: knowledge cutoff, hallucination on private data, factual grounding.

**The pipeline**

**Ingestion (offline)**
```
Documents → Chunking → Embedding → Vector Store
```

**Query (online)**
```
User Query → Embed → Vector Search → Retrieve Chunks → Augment Prompt → LLM → Response
```

**Chunking strategies**
```csharp
// Fixed size
var chunks = text.Chunk(512).Select(chars => new string(chars)).ToList();

// Sentence-based (better for coherence)
var sentences = text.Split(new[] {'.', '!', '?'}, StringSplitOptions.TrimEntries);
var chunks = GroupIntoChunks(sentences, maxTokens: 512, overlap: 50);

// Recursive character splitting (most common)
// Try splitting by paragraph, then sentence, then word, then character
```

**Overlap**
Maintain N tokens of overlap between chunks to preserve context at boundaries.

**Embedding generation in .NET**
```csharp
// Using Semantic Kernel
var kernel = Kernel.CreateBuilder()
    .AddOpenAITextEmbeddingGeneration("text-embedding-3-small", apiKey)
    .Build();

var embeddingService = kernel.GetRequiredService<ITextEmbeddingGenerationService>();
var embedding = await embeddingService.GenerateEmbeddingAsync(text);
float[] vector = embedding.ToArray();  // 1536 dimensions for text-embedding-3-small
```

**Vector search with Qdrant**
```csharp
var qdrantClient = new QdrantClient("localhost");

// Create collection
await qdrantClient.CreateCollectionAsync("documents",
    new VectorParams { Size = 1536, Distance = Distance.Cosine });

// Index a document
await qdrantClient.UpsertAsync("documents", new[]
{
    new PointStruct
    {
        Id = new PointId { Uuid = Guid.NewGuid().ToString() },
        Vectors = vector,
        Payload = { ["text"] = chunk, ["source"] = documentPath }
    }
});

// Search
var results = await qdrantClient.SearchAsync("documents",
    queryVector: queryEmbedding,
    limit: 5,
    scoreThreshold: 0.7f);

var relevantChunks = results.Select(r => r.Payload["text"].StringValue).ToList();
```

**Augmented prompt**
```csharp
var context = string.Join("\n\n", relevantChunks);

var messages = new[]
{
    new ChatMessage { Role = "system", Content = $"""
        You are a helpful assistant. Answer questions based on the provided context.
        If the answer is not in the context, say "I don't have that information."
        
        Context:
        {context}
        """ },
    new ChatMessage { Role = "user", Content = userQuestion }
};
```

**Hybrid search**
Combine vector similarity with keyword search (BM25) for better recall:
```
final_score = alpha × vector_score + (1 - alpha) × keyword_score
```

**RAG evaluation metrics**
- **Faithfulness**: Is the answer supported by the retrieved context? (no hallucination)
- **Answer relevancy**: Is the answer relevant to the question?
- **Context precision**: Are the retrieved chunks actually relevant?
- **Context recall**: Were all relevant chunks retrieved?

## 22.04 — Semantic Kernel in .NET

**What it is**
Microsoft's official .NET SDK for building AI applications. Provides: plugin system, planner, memory, multi-model support.

**Setup**
```csharp
var builder = Kernel.CreateBuilder();
builder.AddOpenAIChatCompletion("gpt-4o", Environment.GetEnvironmentVariable("OPENAI_KEY")!);
builder.AddOpenAITextEmbeddingGeneration("text-embedding-3-small", apiKey);

// Add plugins
builder.Plugins.AddFromType<OrderPlugin>();
builder.Plugins.AddFromType<InventoryPlugin>();

var kernel = builder.Build();
```

**Plugins (native functions)**
```csharp
public class OrderPlugin
{
    private readonly IOrderRepository _orders;
    
    [KernelFunction("get_order")]
    [Description("Get order details by order ID")]
    public async Task<string> GetOrderAsync(
        [Description("The order ID to look up")] int orderId)
    {
        var order = await _orders.GetByIdAsync(orderId);
        return order is null
            ? "Order not found"
            : JsonSerializer.Serialize(new { order.Id, order.Status, order.Total });
    }
    
    [KernelFunction("list_recent_orders")]
    [Description("Get the most recent orders for a customer")]
    public async Task<string> ListRecentOrdersAsync(
        [Description("Customer ID")] int customerId,
        [Description("Number of orders to return")] int count = 5)
    {
        var orders = await _orders.GetRecentAsync(customerId, count);
        return JsonSerializer.Serialize(orders.Select(o => new { o.Id, o.Status, o.Total }));
    }
}
```

**Function calling (tool use)**
```csharp
// Enable automatic function calling
var settings = new OpenAIPromptExecutionSettings
{
    ToolCallBehavior = ToolCallBehavior.AutoInvokeKernelFunctions
};

var result = await kernel.InvokePromptAsync(
    "What is the status of order 42?",
    new KernelArguments(settings));

Console.WriteLine(result.GetValue<string>());
// Model calls get_order(42), gets result, formats response
```

**Chat with history**
```csharp
var chatService = kernel.GetRequiredService<IChatCompletionService>();
var history = new ChatHistory();
history.AddSystemMessage("You are a helpful customer service agent for our e-commerce platform.");

while (true)

    Console.WriteLine(assistantMessage);
}
```

## 22.05 — Production AI Engineering

**Latency management**
```csharp
// Streaming responses
await foreach (var chunk in chatService.GetStreamingChatMessageContentsAsync(history))
{
    Console.Write(chunk.Content);  // Stream to client as Server-Sent Events
    await context.Response.WriteAsync(chunk.Content ?? "");
    await context.Response.Body.FlushAsync();

// Timeouts
using var cts = new CancellationTokenSource(TimeSpan.FromSeconds(30));
try
{
    var result = await GetLlmResponseAsync(prompt, cts.Token);
}
catch (OperationCanceledException)
{
    return "Request timed out — please try again";
}
```

**Retry with exponential backoff**
```csharp
var pipeline = new ResiliencePipelineBuilder()
    .AddRetry(new RetryStrategyOptions
    {
        MaxRetryAttempts = 3,
        Delay = TimeSpan.FromSeconds(1),
        BackoffType = DelayBackoffType.Exponential,
        ShouldHandle = new PredicateBuilder()
            .Handle<RateLimitException>()
            .Handle<HttpRequestException>()
    })
    .Build();

var result = await pipeline.ExecuteAsync(async ct =>
    await GetLlmResponseAsync(prompt, ct));
```

**Output validation**
```csharp
public async Task<ExtractedData?> ExtractWithValidationAsync(string text, int maxRetries = 2)
{
    for (int attempt = 0; attempt <= maxRetries; attempt++)
    {
        var response = await GetStructuredResponseAsync(text);
        
        try
        {
            var data = JsonSerializer.Deserialize<ExtractedData>(response);
            var validationResult = await _validator.ValidateAsync(data!);
            
            if (validationResult.IsValid) return data;
            
            // On failure, add the error to the prompt and retry
            text += $"\n\nPrevious attempt failed validation: {validationResult.ToString()}. Please fix and try again.";
        }
        catch (JsonException)
        {
            text += "\n\nPrevious response was not valid JSON. Return only valid JSON.";
        }
    }
    return null;  // All retries exhausted
}
```
