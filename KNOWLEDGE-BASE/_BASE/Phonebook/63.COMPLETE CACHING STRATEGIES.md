## 63.01 — HTTP Cache Headers

**Cache-Control**
```csharp
// In ASP.NET Core
[ResponseCache(Duration = 300, Location = ResponseCacheLocation.Any, VaryByHeader = "Accept")]
public async Task<IActionResult> GetProducts()
{
    return Ok(await _productService.GetAllAsync());
}

// Cache-Control: max-age=300, public
// Tells: client AND intermediaries (CDN, proxy) can cache for 300 seconds

// Or manually:
context.Response.Headers.CacheControl = "no-cache, no-store, must-revalidate";
context.Response.Headers.Pragma = "no-cache";
context.Response.Headers.Expires = "0";

// Directives:
// public: any cache can store it
// private: only browser cache (not CDN)
// no-cache: must revalidate with server before using cached copy
// no-store: never cache
// max-age=N: cache for N seconds
// s-maxage=N: CDN cache for N seconds (overrides max-age for shared caches)
// stale-while-revalidate=N: serve stale while refreshing in background
// immutable: never revalidate (for versioned assets: app.abc123.js)
```

**ETags (conditional requests)**
```csharp
// Generate ETag from data version
app.MapGet("/api/products/{id}", async (int id, AppDbContext db, HttpContext ctx, CancellationToken ct) =>
{
    var product = await db.Products.AsNoTracking().FirstOrDefaultAsync(p => p.Id == id, ct);
    if (product is null) return Results.NotFound();
    
    // ETag based on RowVersion (changes only when data changes)
    var etag = $"\"{Convert.ToBase64String(product.RowVersion)}\"";
    
    // Check If-None-Match header (client's cached ETag)
    if (ctx.Request.Headers.IfNoneMatch.Contains(etag))
        return Results.StatusCode(304);  // Not Modified — client's cache is valid
    
    ctx.Response.Headers.ETag = etag;
    ctx.Response.Headers.CacheControl = "max-age=60, must-revalidate";
    return Results.Ok(product.ToDto());
});
```

**CDN integration**
```
Static assets (JS, CSS, images):
  Cache-Control: public, max-age=31536000, immutable
  (1 year, immutable — filename includes content hash like app.abc123.js)

API responses:
  Public data: Cache-Control: public, max-age=60, s-maxage=300
  User-specific data: Cache-Control: private, max-age=30
  Never cache: Cache-Control: no-store
```

## 63.02 — Distributed Cache Patterns

**Cache-aside with stampede prevention**
```csharp
public class CacheService
{
    private readonly IDistributedCache _cache;
    private static readonly SemaphoreSlim _lock = new(1, 1);
    
    public async Task<T?> GetOrSetAsync<T>(
        string key,
        Func<CancellationToken, Task<T>> factory,
        TimeSpan ttl,
        CancellationToken ct = default)
    {
        // Fast path: check cache without lock
        var cached = await _cache.GetStringAsync(key, ct);
        if (cached is not null)
            return JsonSerializer.Deserialize<T>(cached);
        
        // Slow path: acquire lock to prevent stampede
        await _lock.WaitAsync(ct);
        try
        {
            // Double-check inside lock
            cached = await _cache.GetStringAsync(key, ct);
            if (cached is not null)
                return JsonSerializer.Deserialize<T>(cached);
            
            // Compute the value
            var value = await factory(ct);
            if (value is not null)
            {
                await _cache.SetStringAsync(
                    key,
                    JsonSerializer.Serialize(value),
                    new DistributedCacheEntryOptions
                    {
                        AbsoluteExpirationRelativeToNow = ttl + 
                            TimeSpan.FromSeconds(Random.Shared.Next(0, 30))  // Jitter to prevent synchronized expiry
                    },
                    ct);
            }
            return value;
        }
        finally
        {
            _lock.Release();
        }
    }
}

// Usage
var product = await _cache.GetOrSetAsync(
    $"product:{productId}",
    ct => _db.Products.AsNoTracking().FirstOrDefaultAsync(p => p.Id == productId, ct),
    TimeSpan.FromMinutes(10));
```

**Write-through cache**
```csharp
// On every write: update both DB and cache simultaneously
public async Task UpdateProductAsync(UpdateProductCommand cmd, CancellationToken ct)
{
    // 1. Update in database
    var product = await _db.Products.FindAsync(cmd.ProductId, ct);
    product!.UpdatePrice(cmd.NewPrice);
    await _db.SaveChangesAsync(ct);
    
    // 2. Update in cache (write-through)
    await _cache.SetStringAsync(
        $"product:{cmd.ProductId}",
        JsonSerializer.Serialize(product.ToDto()),
        new DistributedCacheEntryOptions { AbsoluteExpirationRelativeToNow = TimeSpan.FromMinutes(10) },
        ct);
    
    // Now reads get fresh data from cache immediately
}
```
