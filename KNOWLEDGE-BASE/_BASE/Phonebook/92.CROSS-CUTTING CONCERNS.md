## 92.01 — Structured Logging Best Practices

**What structured logging is**
```csharp
// Unstructured (plain text — hard to query)
_logger.LogInformation("User 42 placed order 123 for $99.99 at 2026-06-01");

// Structured (named properties — queryable in Grafana, Seq, Kibana)
_logger.LogInformation(
    "User {UserId} placed order {OrderId} for {Amount:C} at {OrderTime}",
    userId, orderId, amount, DateTime.UtcNow);
// Serilog/MS Logging captures: UserId=42, OrderId=123, Amount=99.99, OrderTime=...
```

**Log levels — when to use each**
```csharp
// TRACE: Extremely verbose. Algorithm step-by-step. Development only.
_logger.LogTrace("Processing item {Index} of {Total}", i, items.Count);

// DEBUG: Useful during development. Diagnosing specific issues.
_logger.LogDebug("Cache miss for key {Key}, fetching from database", cacheKey);

// INFORMATION: Normal operation. Business events. High-level workflow.
_logger.LogInformation("Order {OrderId} created by user {UserId}", orderId, userId);
_logger.LogInformation("Payment {PaymentId} succeeded for {Amount:C}", paymentId, amount);

// WARNING: Something unexpected but handled. Potential problem.
_logger.LogWarning("Retry attempt {Attempt} for order {OrderId}", attempt, orderId);
_logger.LogWarning("Rate limit approaching for user {UserId}: {Current}/{Limit}", userId, current, limit);

// ERROR: Something went wrong. Requires investigation. NOT immediate outage.
_logger.LogError(ex, "Failed to send email to {Email} for order {OrderId}", email, orderId);
_logger.LogError("Payment failed for order {OrderId}: {Reason}", orderId, reason);

// CRITICAL: Immediate action required. System-wide failure. Wake someone up.
_logger.LogCritical("Database connection lost. All requests failing.");
```

**Log enrichment**
```csharp
// Serilog enrichers — add context to all log entries
Log.Logger = new LoggerConfiguration()
    .Enrich.FromLogContext()          // Push/pop context
    .Enrich.WithMachineName()
    .Enrich.WithEnvironmentUserName()
    .Enrich.WithThreadId()
    .Enrich.WithProperty("Application", "MyApp")
    .Enrich.WithProperty("Version", "1.2.3")
    .WriteTo.Seq("http://localhost:5341")
    .CreateLogger();

// Per-request enrichment (correlation ID, user ID)
app.Use(async (context, next) =>
{
    using (LogContext.PushProperty("CorrelationId",
        context.Request.Headers["X-Correlation-Id"].FirstOrDefault() ?? Guid.NewGuid().ToString()))
    using (LogContext.PushProperty("UserId",
        context.User?.FindFirstValue(ClaimTypes.NameIdentifier)))
    using (LogContext.PushProperty("RequestPath", context.Request.Path))
    {
        await next();
    }
});
// Now every log entry in this request has CorrelationId and UserId automatically
```

## 92.02 — Distributed Tracing Implementation

**OpenTelemetry complete setup**
```csharp
builder.Services.AddOpenTelemetry()
    .ConfigureResource(resource => resource
        .AddService(
            serviceName: "MyApp.Api",
            serviceVersion: "1.0.0",
            serviceInstanceId: Environment.MachineName))
    .WithTracing(tracing => tracing
        .AddAspNetCoreInstrumentation(options =>
        {
            options.RecordException = true;
            options.Filter = httpContext =>
                !httpContext.Request.Path.StartsWithSegments("/health");  // Skip health checks
        })
        .AddHttpClientInstrumentation(options =>
        {
            options.RecordException = true;
            options.FilterHttpRequestMessage = req =>
                req.RequestUri?.Host != "metadata.google.internal";  // Skip cloud metadata
        })
        .AddEntityFrameworkCoreInstrumentation(options =>
        {
            options.SetDbStatementForText = true;  // Log SQL (don't in prod if sensitive)
        })
        .AddRedisInstrumentation()
        .AddSource("MyApp.*")  // Custom activities
        .AddJaegerExporter(options =>
        {
            options.AgentHost = builder.Configuration["Jaeger:Host"] ?? "localhost";
            options.AgentPort = 6831;
        }))
    .WithMetrics(metrics => metrics
        .AddAspNetCoreInstrumentation()
        .AddHttpClientInstrumentation()
        .AddRuntimeInstrumentation()  // GC, thread pool, memory
        .AddPrometheusExporter());

// Custom activity source for business operations
public static class AppTracing
{
    public static readonly ActivitySource Source = new("MyApp.Orders");
}

// Create custom spans in business code
public async Task ProcessOrderAsync(int orderId, CancellationToken ct)
{
    using var activity = AppTracing.Source.StartActivity("ProcessOrder");
    activity?.SetTag("order.id", orderId);
    
    try
    {
        await DoWork(orderId, ct);
        activity?.SetStatus(ActivityStatusCode.Ok);
    }
    catch (Exception ex)
    {
        activity?.SetStatus(ActivityStatusCode.Error, ex.Message);
        activity?.RecordException(ex);
        throw;
    }
}
```
