## 50.01 — Essential .NET Tools

```bash
# .NET CLI
dotnet new webapi -n MyProject          # Create new project
dotnet build                            # Build
dotnet run                             # Run
dotnet test                            # Run tests
dotnet publish -c Release              # Publish
dotnet add package <PackageName>       # Add NuGet package
dotnet list package --outdated         # Check for updates

# EF Core tools
dotnet tool install --global dotnet-ef
dotnet ef migrations add <Name>
dotnet ef database update
dotnet ef migrations script

# Performance tools
dotnet tool install --global dotnet-counters
dotnet counters monitor --process-id <PID>     # Live metrics
dotnet counters collect --process-id <PID>     # Record metrics

dotnet tool install --global dotnet-trace
dotnet trace collect --process-id <PID>        # CPU profiling
dotnet trace report <tracefile>                # Analyze

dotnet tool install --global dotnet-dump
dotnet dump collect --process-id <PID>         # Memory dump
dotnet dump analyze <dumpfile>                 # Analyze dump

dotnet tool install --global dotnet-gcdump
dotnet gcdump collect --process-id <PID>       # GC heap dump

# Code quality
dotnet format                                  # Format code
dotnet analyzers                               # Run Roslyn analyzers
```

## 50.03 — Roslyn Analyzers and Code Quality

```xml
<!-- In .csproj — strict analysis -->
<PropertyGroup>
  <Nullable>enable</Nullable>
  <ImplicitUsings>enable</ImplicitUsings>
  <TreatWarningsAsErrors>true</TreatWarningsAsErrors>
  <AnalysisLevel>latest-recommended</AnalysisLevel>
  <EnableNETAnalyzers>true</EnableNETAnalyzers>
</PropertyGroup>

<!-- Additional analyzers -->
<ItemGroup>
  <PackageReference Include="StyleCop.Analyzers" Version="1.2.0-beta.556" PrivateAssets="all" />
  <PackageReference Include="SonarAnalyzer.CSharp" Version="9.24.0.89429" PrivateAssets="all" />
  <PackageReference Include="Roslynator.Analyzers" Version="4.12.1" PrivateAssets="all" />
  <PackageReference Include="Microsoft.CodeAnalysis.NetAnalyzers" Version="8.0.0" PrivateAssets="all" />
</ItemGroup>
```
